# Color in Architecture

## Overview
Color transforms architecture from form to experience. It affects perceived room size, emotional response, and building character. Understanding color is essential for both exterior facades and interior design.

## Color Psychology in Architecture

| Color Family | Psychological Effect | Architectural Use |
|-------------|---------------------|-------------------|
| **White** | Purity, spaciousness, light | Modernist facades, small rooms |
| **Grey** | Sophistication, neutrality | Contemporary, industrial |
| **Black** | Drama, depth, elegance | Accents, dark interiors |
| **Blue** | Calm, trust, coolness | Bedrooms, bathrooms, offices |
| **Green** | Nature, health, balance | Biophilic interiors, hospitals |
| **Yellow** | Energy, warmth, optimism | Kitchens, children's spaces |
| **Red** | Energy, warmth, appetite | Dining rooms, accents only |
| **Earth tones** | Warmth, grounding, natural | Timber cladding, stone, plaster |
| **Terracotta** | Mediterranean warmth | Facade, flooring |

## Color and Space Perception

| Technique | Effect | Application |
|-----------|--------|-------------|
| Light colors | Room feels larger | Small rooms, low ceilings |
| Dark colors | Room feels more intimate | Large rooms, feature walls |
| Warm colors | Surfaces feel closer | Cozy, enclosed feel |
| Cool colors | Surfaces recede | Spacious, airy feel |
| Accent wall | Adds depth without closing | One colored wall, three neutral |
| Monochrome | Unity and sophistication | Same tone, varied materials |
| Complementary | Dynamic contrast | Orange/blue, green/red accents |

## Exterior Color Considerations

| Factor | Consideration | Notes |
|--------|--------------|-------|
| Climate | Light colors reflect heat | Important in hot climates |
| Context | Relate to surroundings | Urban vs. rural, neighbors |
| Material | Color inherent in material | Timber, stone, brick |
| Maintenance | Lighter colors show dirt more | Consider cleaning access |
| Regulations | Some areas restrict colors | Check PUG/PUZ requirements |
| Aging | Materials patina differently | Timber silvers, copper greens |

## ModulCA Application
- **Exterior cladding**: Timber (natural/stained), fibre-cement (painted), metal (any RAL color)
- **Brand colors**: Brand-teal-800 (primary) and brand-amber-500 (accent)
- **Module mix**: Different cladding colors per module for visual interest
- **Interior**: Each module can have independent color scheme
- **NCS/RAL system**: Standard color references for factory finishing
