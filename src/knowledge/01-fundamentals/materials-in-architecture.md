# Materials in Architecture — Properties & Character

## Overview
Materials are the physical vocabulary of architecture. Each material has inherent properties — structural, thermal, visual, tactile — that determine how it can and should be used. Choosing materials is both a technical and an aesthetic decision.

## Primary Architectural Materials

### Timber (Wood)
| Property | Character |
|----------|-----------|
| Structure | Tension + compression, warm, light |
| Thermal | Natural insulator (λ = 0.13 W/mK) |
| Acoustic | Warm, absorptive |
| Visual | Grain, warmth, natural variation |
| Tactile | Warm to touch |
| Sustainability | Renewable, carbon-storing |
| Aging | Silvers outdoors, deepens indoors |

### Stone
| Property | Character |
|----------|-----------|
| Structure | Excellent compression, heavy |
| Thermal | High thermal mass (stores heat) |
| Visual | Permanence, gravity, texture |
| Tactile | Cool to touch |
| Sustainability | Durable but extraction-intensive |
| Types | Granite, limestone, marble, sandstone, slate |

### Concrete
| Property | Character |
|----------|-----------|
| Structure | Compression champion, moldable |
| Thermal | High thermal mass |
| Visual | Raw (brutalist) or refined (fair-faced) |
| Tactile | Cool, hard |
| Sustainability | High CO₂ from cement production |
| Forms | In-situ, precast, fiber-reinforced |

### Steel
| Property | Character |
|----------|-----------|
| Structure | Tension + compression, slender elements |
| Thermal | Conductor (thermal bridge risk) |
| Visual | Precision, industrial, minimal |
| Sustainability | Recyclable but energy-intensive to produce |
| Forms | Hot-rolled sections, cold-formed, cables |

### Glass
| Property | Character |
|----------|-----------|
| Structure | Compression only (structurally limited) |
| Thermal | Poor insulator (unless multi-pane) |
| Visual | Transparency, lightness, views |
| Types | Float, tempered, laminated, insulated, low-E |
| Innovation | Smart glass, electrochromic, BIPV |

### Brick
| Property | Character |
|----------|-----------|
| Structure | Good compression, modular unit |
| Thermal | Good thermal mass |
| Visual | Texture, scale, pattern, color range |
| Tactile | Warm, textured |
| Sustainability | Durable, reusable, moderate embodied energy |

## Material Selection Criteria

| Criterion | Questions to Ask |
|-----------|-----------------|
| Structural | Can it carry the loads? Span the distances? |
| Thermal | Does it insulate or store heat? |
| Fire | How does it perform in fire? Rating? |
| Durability | How long will it last? Maintenance needs? |
| Sustainability | Embodied carbon? Recyclable? Renewable? |
| Cost | Material + labor + maintenance over lifecycle? |
| Aesthetics | Color, texture, pattern, aging character? |
| Availability | Local sourcing? Transport cost? |
| Buildability | Skill requirements? Equipment? |

## ModulCA Application
ModulCA primarily uses:
- **CLT** (structure) — warm, sustainable, carbon-negative
- **Mineral wool** (insulation) — fire-resistant, excellent thermal
- **Gypsum board** (interior finish) — smooth, paintable, fire-rated
- **Timber / fibre-cement / metal** (cladding) — client choice for character
- **Triple-glazed windows** — thermal performance with views
