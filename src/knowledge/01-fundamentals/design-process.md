# The Architectural Design Process

## Overview
Architecture follows a structured design process from initial brief to completed building. Understanding these stages helps clients collaborate effectively with architects and make decisions at the right time.

## Design Stages (Based on RIBA Plan of Work)

| Stage | Name | Key Activities | Decisions |
|-------|------|---------------|-----------|
| 0 | **Strategic Definition** | Define project goals, budget, site | Go/no-go |
| 1 | **Preparation & Briefing** | Client brief, site analysis, feasibility | Requirements confirmed |
| 2 | **Concept Design** | Massing, layout options, style direction | Concept approved |
| 3 | **Spatial Coordination** | Detailed plans, structural + MEP coordination | Layout locked |
| 4 | **Technical Design** | Construction details, specifications, permits | Ready for construction |
| 5 | **Manufacturing & Construction** | Factory production + site assembly | Build |
| 6 | **Handover** | Completion, inspection, occupancy | Move in |
| 7 | **Use** | Maintenance, adaptation, evaluation | Ongoing |

## Client's Role at Each Stage

| Stage | Client Input | Key Questions to Answer |
|-------|-------------|----------------------|
| 0-1 | Budget, lifestyle needs, site info | How much? How many rooms? Where? |
| 2 | Feedback on options, preferences | Which layout? Which style? |
| 3 | Approve detailed plans | Room sizes OK? Flow works? |
| 4 | Approve specifications, materials | Finishes? Fixtures? Colors? |
| 5 | Site visits (optional) | Progress checks |
| 6 | Walkthrough, snag list | Everything working? |
