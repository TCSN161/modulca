# Form, Space & Order — Fundamental Design Elements

## Overview
Based on Francis D.K. Ching's seminal work, these are the building blocks of architectural composition. Every building is a composition of forms defining spaces, organized through ordering principles.

## Primary Forms

| Form | Character | Architectural Use |
|------|-----------|-------------------|
| **Circle/Cylinder** | Unity, continuity | Rotundas, towers, domes |
| **Square/Cube** | Stability, neutrality | Rooms, modules, pavilions |
| **Triangle/Pyramid** | Stability, direction | Roofs, trusses, A-frames |
| **Rectangle/Prism** | Most common building form | Rooms, corridors, buildings |

The 3×3m ModulCA module is a **square** in plan — the most neutral and combinable of all primary forms.

## Space-Defining Elements

| Element | How It Defines Space | Example |
|---------|---------------------|---------|
| **Base plane** | Elevation change defines zones | Split-level, raised platform |
| **Overhead plane** | Ceiling/canopy creates enclosure | Lower ceiling = intimate |
| **Wall planes** | Vertical enclosure, varying degrees | Full wall = private, half wall = connected |
| **Openings** | Connect spaces, frame views | Windows, doors, archways |
| **Columns** | Define space without enclosing | Colonnade, pilotis |

## Ordering Principles

| Principle | Description | Application |
|-----------|-------------|-------------|
| **Axis** | Imaginary line organizing elements | Entry axis, garden vista |
| **Symmetry** | Balanced arrangement around axis | Classical facades |
| **Hierarchy** | Size/placement shows importance | Large living room, smaller bedrooms |
| **Datum** | Line/plane that gathers elements | Continuous wall, floor level |
| **Rhythm** | Regular pattern of elements | Repeated modules, window spacing |
| **Repetition** | Same element used multiple times | Identical modules |
| **Transformation** | Gradual change in form | Tapering, rotating modules |

## Spatial Relationships

| Relationship | Description | Module Application |
|-------------|-------------|-------------------|
| **Space within space** | Smaller space inside larger | Bathroom pod within open plan |
| **Interlocking** | Overlapping volumes | Shared walls between modules |
| **Adjacent** | Side by side, shared wall | Standard module connection |
| **Linked** | Connected by intermediate element | Corridor or gallery between modules |

## ModulCA Application
ModulCA's modular system naturally expresses these principles:
- **Repetition**: Identical 3×3m modules create rhythm
- **Hierarchy**: Combine more modules for important rooms
- **Axis**: Entry module defines the primary axis
- **Interlocking**: Shared walls create flowing open spaces
- **Transformation**: L-shape, T-shape, courtyard — same module, different orders
