# A Pattern Language — Design Patterns for Architecture

## Overview
Christopher Alexander's "A Pattern Language" (1977) identified 253 architectural and urban design patterns — recurring solutions to design problems that create livable, human-centered spaces. The patterns range from city-scale down to construction details.

## Key Patterns Relevant to Residential Design

### Patterns for the Site (Large Scale)

| Pattern # | Name | Description |
|-----------|------|-------------|
| 37 | **House Cluster** | Groups of 8-12 houses sharing common land |
| 38 | **Row Houses** | Connected houses forming a street |
| 104 | **Site Repair** | Build on worst part of site, preserve best |
| 105 | **South Facing Outdoors** | Place outdoor spaces to catch sun |
| 106 | **Positive Outdoor Space** | Outdoor spaces enclosed on sides, not leftover |
| 107 | **Wings of Light** | Long narrow buildings for daylight penetration |

### Patterns for the House (Medium Scale)

| Pattern # | Name | Description |
|-----------|------|-------------|
| 110 | **Main Entrance** | Visible, welcoming, transition from public to private |
| 127 | **Intimacy Gradient** | Public → semi-private → private spaces in sequence |
| 129 | **Common Areas at Heart** | Living/kitchen at center of house |
| 130 | **Entrance Room** | Transition space between outside and inside |
| 131 | **Flow Through Rooms** | Rooms connect to each other, not just to corridor |
| 133 | **Staircase as Stage** | Stairs as social/visual element, not hidden |
| 134 | **Zen View** | A special view revealed at a specific point |
| 137 | **Children's Realm** | Children have their own connected spaces |
| 141 | **A Room of One's Own** | Every person needs a private retreat |
| 142 | **Sequence of Sitting Spaces** | Multiple scales of sitting: alone, pair, group |
| 159 | **Light on Two Sides** | Every room has windows on at least two walls |

### Patterns for Construction (Small Scale)

| Pattern # | Name | Description |
|-----------|------|-------------|
| 197 | **Thick Walls** | Deep walls with niches, seats, shelves |
| 201 | **Waist-High Shelf** | A continuous shelf at waist height in rooms |
| 202 | **Built-In Seats** | Window seats, bench corners |
| 203 | **Child Caves** | Small spaces for children to play and hide |
| 222 | **Low Sill** | Window sills at 12-14 inches for seated views |
| 236 | **Windows Which Open Wide** | Full opening for fresh air and connection |
| 250 | **Warm Colors** | Earthy, warm tones for interior finishes |

## The Pattern Language Concept
Patterns are not rigid rules — they are principles that can be combined:
- Each pattern solves a specific problem
- Patterns reference other related patterns (like hyperlinks)
- A "language" is your selection of patterns for a specific project
- No two buildings using the same patterns will be identical

## ModulCA Application
The modular system naturally supports many Alexander patterns:
- **#107 Wings of Light**: 3m module width ensures daylight penetration
- **#127 Intimacy Gradient**: Module layout from entry → living → bedrooms
- **#129 Common Areas at Heart**: Central open-plan module cluster
- **#141 A Room of One's Own**: Each module can be a private retreat
- **#159 Light on Two Sides**: Corner modules achieve this; L-shapes create opportunities
