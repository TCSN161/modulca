# Organic Architecture

## Overview
Organic architecture, championed by Frank Lloyd Wright, seeks harmony between human habitation and nature. A building should grow from its site like a plant — responding to landscape, climate, and the way people live. It's not about copying natural forms but about applying nature's principles of integration, growth, and adaptation.

## Key Principles (Wright)

| Principle | Description | Example |
|-----------|-------------|---------|
| Nature integration | Building belongs to its landscape | Fallingwater (built over waterfall) |
| Horizontal emphasis | Low, spreading forms echo the earth | Prairie houses |
| Natural materials | Use materials from the region | Local stone, timber |
| Open plan | Flowing, interconnected spaces | Destroy the box |
| Inside-outside | Blurred boundaries with nature | Cantilevers, terraces, large glazing |
| Human scale | Compressed entry, expanded living | Low ceiling at door → high ceiling inside |
| Organic geometry | Non-rectilinear when appropriate | Hexagonal, circular plans |

## Organic Design for Residential

| Element | Organic Approach |
|---------|-----------------|
| Site response | Study sun, wind, views, trees, topography before design |
| Entrance | Compressed entry path → expanding into light-filled interior |
| Living areas | Flow toward the best view / garden |
| Roof | Low-pitched or green roof — blends into landscape |
| Materials | Match the site — timber in forests, stone in hills |
| Landscape | Building and garden are one design — no separation |

## ModulCA Application
- **Site response**: Module layout responds to plot — L toward garden, opening toward sun
- **Landscape integration**: Modules on screw piles have minimal site disturbance — organic approach
- **Green roof**: Modules with sedum/green roofs blend into landscape
- **Natural materials**: CLT and timber cladding belong in natural settings
- **Flow**: Open-plan module combinations create Wright-like flowing spaces
- **Compressed entry**: Narrow entrance module → open into double-height living space
