# Japanese Design & Wabi-Sabi

## Overview
Japanese architecture and design philosophy values impermanence, simplicity, and the beauty of natural imperfection. "Wabi-sabi" finds beauty in the worn, weathered, and incomplete. "Ma" (間) is the conscious use of empty space. Together, they create architecture that is deeply connected to nature, mindful, and serene.

## Key Concepts

| Concept | Meaning | Architectural Application |
|---------|---------|---------------------------|
| Wabi-sabi | Beauty in imperfection, transience | Weathered wood, patinated metals, natural aging |
| Ma (間) | Meaningful emptiness, negative space | Open floor areas, uncluttered rooms, pauses |
| Engawa | Transitional space (veranda) | Indoor-outdoor threshold, covered terrace |
| Shoji | Translucent sliding screens | Diffused light, flexible partitions |
| Tokonoma | Alcove for art/contemplation | Display niche, focal point in room |
| Genkan | Entrance step-down/transition | Clear threshold between outside and inside |

## Japanese Interior Characteristics

| Feature | Description |
|---------|-------------|
| Natural materials | Wood, bamboo, paper, stone, clay |
| Flexible spaces | Sliding panels instead of fixed walls |
| Floor-level living | Low furniture, tatami, floor cushions |
| Indirect light | Shoji screens, diffused natural light |
| Nature framing | Windows as paintings of the garden |
| Minimal objects | Few, carefully chosen items |
| Seasonal change | Interiors change with seasons (flowers, textiles) |

## Wabi-Sabi Material Palette

| Material | Wabi-Sabi Quality | Application |
|----------|--------------------|-------------|
| Shou sugi ban (charred wood) | Blackened, textured, preserved | Exterior cladding |
| Lime plaster | Soft, imperfect, breathable | Interior walls |
| Raw concrete | Formwork marks, aging | Foundation, accent wall |
| Oxidized steel (Corten) | Rust patina, changes over time | Garden elements, screens |
| Natural stone (rough) | Unpolished, organic | Entrance, bathroom |
| Handmade ceramics | Irregular, unique | Bathroom basins, tile |

## ModulCA Application
- **Shou sugi ban cladding**: Charred timber exterior — Japanese technique, fire-resistant, zero maintenance
- **Engawa module**: One module as a covered veranda — traditional Japanese threshold space
- **Sliding module walls**: Replace fixed wall with sliding panels — flexible Japanese space
- **Ma in layout**: Leave deliberate "empty" modules — covered outdoor space, courtyard
- **CLT aging**: Allow timber to develop natural patina — wabi-sabi beauty over time
- **Genkan entrance**: Step-up into module (screw piles) creates a natural genkan transition
- **Garden framing**: Position module windows to frame specific garden views — Japanese "borrowed scenery"
