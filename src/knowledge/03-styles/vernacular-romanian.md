# Romanian Vernacular Architecture

## Overview
Romania has one of Europe's richest traditions of vernacular architecture — timber houses that evolved over centuries in response to climate, materials, and ways of life. From the Maramureș wooden churches (UNESCO) to the Transylvanian fortified churches, Romanian folk architecture demonstrates sophisticated construction and deep connection to landscape.

## Regional Traditions

| Region | Characteristic | Key Feature |
|--------|---------------|-------------|
| Maramureș | Tall timber houses, ornate gates | Carved wooden gates, shingle roofs |
| Transylvania | Saxon fortified houses, wide streets | Thick walls, defensive character |
| Wallachia | Low houses, wide prispa (veranda) | Deep porch, hipped roof |
| Moldavia | Painted exteriors, modest scale | Decorative plaster, Byzantine influence |
| Oltenia | Earth-walled houses, blue accents | Vernacular color traditions |
| Dobrogea | Stone/earth, compact, wind-resistant | Flat roofs, courtyard plans |

## Common Elements Across Regions

| Element | Description | Modern Equivalent |
|---------|-------------|-------------------|
| Prispa (veranda) | Covered porch along one or more facades | Covered terrace module |
| Tinda (entrance hall) | Transitional space between outside and in | Entrance module/airlock |
| Sobă (masonry stove) | Central thermal mass for heating | Thermal mass wall, heat pump |
| Timber frame | Hand-hewn log or timber stud walls | CLT, engineered timber |
| Steep roof | Shingle or tile, 45-60° pitch | Green roof or standing seam |
| Carved decoration | Window frames, gates, columns | CNC-cut decorative panels |
| Courtyard | Rural houses oriented around yard | Module courtyard configuration |
| Root cellar | Below-ground cool storage | Utility module |

## Traditional Proportions

| Element | Traditional Measure | Modern Equivalent |
|---------|-------------------|-------------------|
| Room size | ~3×4m (one stânjeni = ~1.96m) | 3×3m module ≈ traditional room |
| Ceiling height | 2.40-2.60m | 2.70m (ModulCA standard) |
| Prispa depth | 1.50-2.00m | 1.50-3.00m terrace module |
| Window proportions | Tall, narrow, shuttered | Vertical emphasis possible |
| Eaves overhang | 0.60-1.00m | Factory-built overhang |

## Maramureș Wooden Architecture (UNESCO)

| Building | Feature | Date |
|----------|---------|------|
| Church of Bârsana | 57m tall wooden spire | 1720 |
| Church of Budești | Painted interior frescoes | 1643 |
| Wooden gates | Carved sun motifs, rope patterns | Ongoing tradition |
| Traditional houses | Two-room plan + prispa + attic | 18th-19th century |

## ModulCA Connection
- **Prispa module**: Dedicated covered veranda module — the most Romanian element
- **Timber construction**: CLT continues the Romanian tradition of building in wood
- **3m ≈ traditional room**: The 3×3m module is remarkably close to traditional Romanian room proportions
- **Carved details**: CNC-cut decorative panels can reference Maramureș carving traditions
- **Courtyard layout**: Rural Romanian houses were organized around courtyards — U-shape ModulCA layouts
- **Local materials**: Timber cladding with larch or spruce — same species used traditionally
- **Modern prispa**: Glass-enclosed veranda module for year-round use — evolved tradition
