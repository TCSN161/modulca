# Mediterranean Architecture

## Overview
Mediterranean architecture draws from the building traditions of Southern Europe, North Africa, and the Levant — regions sharing hot, dry summers and mild winters. Key features include thick walls for thermal mass, courtyards for microclimate control, and seamless indoor-outdoor living. It's one of the world's most climate-responsive architectural traditions.

## Key Characteristics

| Feature | Description | Climate Purpose |
|---------|-------------|-----------------|
| White/light walls | Lime-washed or painted white | Reflects solar radiation |
| Thick walls (40-60cm) | Stone, adobe, or plastered masonry | Thermal mass — cool by day, warm by night |
| Courtyard | Central open-air room | Microclimate, ventilation, privacy |
| Terracotta roof tiles | Red clay tiles | Ventilated, durable, regional identity |
| Pergola/loggia | Covered outdoor area | Shade for outdoor living |
| Small windows (north/west) | Minimal glazing on hot sides | Reduces solar heat gain |
| Large openings (courtyard) | Generous courtyard-facing openings | Cross-ventilation |
| Outdoor rooms | Terraces, patios, courtyards | Living space extends outside |

## Courtyard Design Principles

| Principle | Application |
|-----------|-------------|
| Orientation | Open to south for winter sun, shaded in summer |
| Proportions | Width ≥ building height for adequate sun penetration |
| Water feature | Central fountain for evaporative cooling + sound |
| Planting | Citrus, jasmine, bougainvillea — shade + fragrance |
| Floor | Permeable paving — terracotta, gravel, stone |
| Privacy | Walls create private outdoor room |
