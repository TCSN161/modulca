# Parametric & Computational Design

## Overview
Parametric design uses algorithms and computational tools to generate architectural forms from sets of rules and parameters. Instead of drawing a shape, the designer defines relationships — and the computer explores thousands of variations. This enables complex, organic forms that would be impossible to design by hand.

## Key Concepts

| Concept | Description | Application |
|---------|-------------|-------------|
| Parametric modeling | Form defined by variable parameters | Change one parameter → entire form updates |
| Generative design | Algorithm produces multiple design options | AI explores thousands of layouts |
| Topology optimization | Remove material where it's not needed | Structural efficiency |
| Voronoi patterns | Nature-inspired cell division geometry | Facades, partitions, structures |
| Agent-based design | Simulated behaviors generate form | Crowd simulation → circulation design |
| Environmental simulation | Sun, wind, energy inform design | Optimal orientation, shading |

## Parametric at Residential Scale

| Application | How It Works | Benefit |
|-------------|-------------|---------|
| Layout optimization | Algorithm tests all module arrangements for sun/view/privacy | Best layout from thousands of options |
| Facade variation | Parametric window sizes based on orientation | Each face optimized for light/heat |
| Structural optimization | Vary member sizes based on actual loads | Less material, lighter structure |
| Shading design | Sun path drives louver angles | Automated seasonal shading |
| Site analysis | Algorithmic terrain, wind, sun analysis | Data-driven design decisions |
