# Brutalism & Neo-Brutalism

## Overview
Brutalism (from French "béton brut" — raw concrete) celebrates the honest expression of materials and structure. Massive, sculptural, and unapologetic, Brutalist buildings make no attempt to be pretty — they aim to be truthful. Once reviled, Brutalism is now experiencing a revival (Neo-Brutalism) as a reaction against bland contemporary design.

## Key Characteristics

| Feature | Description |
|---------|-------------|
| Raw concrete (béton brut) | Unfinished, showing formwork texture |
| Massive forms | Heavy, sculptural, monolithic |
| Honest structure | Structure IS the architecture |
| Repetition | Modular units expressed on facade |
| Texture | Board-marked concrete, rough surfaces |
| Bold geometry | Angular, geometric, dramatic |
| Social purpose | Originally for public housing, civic buildings |

## Brutalism in Romania
Romania has significant Brutalist heritage from the communist era:

| Building | Location | Notes |
|----------|----------|-------|
| Casa Poporului (Palace of Parliament) | Bucharest | World's heaviest building |
| Communist-era housing blocks | Nationwide | Mass housing in concrete panels |
| Various civic buildings | Major cities | Theaters, cultural centers |
| Sala Palatului | Bucharest | Concert hall, 1960 |

## Neo-Brutalist Residential Design

| Element | Neo-Brutalist Approach |
|---------|----------------------|
| Material | Exposed concrete + warm wood contrast |
| Kitchen | Concrete countertop, open shelving |
| Bathroom | Cast concrete basin, terrazzo floor |
| Living room | Board-formed concrete feature wall |
| Facade | Raw CLT as "timber brutalism" |
