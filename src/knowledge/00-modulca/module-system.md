# The 3×3m Module System

## Overview
Every ModulCA home is built from standardized 3.00 × 3.00m modules. This simple grid creates infinite possibilities through combination and configuration.

## Module Dimensions

| Measurement | Value | Notes |
|-------------|-------|-------|
| Exterior dimensions | 3.00 × 3.00m | Module edge to edge |
| Gross floor area | 9.00 m² | Per module |
| Exterior wall thickness | 0.15m (150mm) | Each side |
| Interior wall thickness | 0.10m (100mm) | Between different rooms |
| Usable interior area | ~5.76 m² (single module) | After exterior walls |
| Shared wall (open plan) | 0mm | Wall removed between same-function modules |
| Usable area (2 modules, shared) | ~14 m² | Shared wall removed |
| Clear ceiling height | 2.70m | Interior |
| Module weight | 2.5–4.0 tonnes | Empty, depending on structure |

## Layout Configurations

### Linear (Line)
```
[M][M][M][M]
```
- Simple row of modules
- Best for: Narrow lots, courtyard walls
- Example: 4 modules = 36m² (28m² usable)

### L-Shape
```
[M][M][M]
[M]
```
- Creates a sheltered corner
- Best for: Garden framing, wind protection
- Example: 4 modules = 36m² with courtyard corner

### T-Shape
```
   [M]
[M][M][M]
```
- Entrance projection or wing
- Best for: Prominent entrance, bedroom wing
- Example: 4 modules with defined entry

### U-Shape
```
[M]   [M]
[M][M][M]
```
- Three-sided courtyard
- Best for: Private garden, wind shelter, south-facing terrace
- Example: 5 modules enclosing an outdoor room

### Courtyard (O-Shape)
```
[M][M][M]
[M]   [M]
[M][M][M]
```
- Enclosed central courtyard
- Best for: Privacy, microclimate, Mediterranean feel
- Example: 8 modules with private garden

### Square / Compact
```
[M][M]
[M][M]
```
- Most efficient footprint
- Best for: Small lots, maximum floor area on minimum land
- Example: 4 modules = 36m² compact house

## Shared Walls
When two modules of the same function are placed adjacent, the shared wall is removed:
- Creates open-plan spaces
- Saves ~€1,500 per shared wall
- Structural frame maintains integrity
- MEP services route through remaining walls

## Two-Storey Construction
- Ground floor modules support upper floor modules
- Stair module provides access
- Upper floor can be smaller than ground (setback)
- Maximum 2-3 stories for light frame without special analysis (P100 seismic)

## Future Expansion
The modular system allows expansion:
- Add modules to existing home later
- Bolt-on connections to existing module edges
- Foundation points pre-planned for future modules
- No demolition or structural modification needed
