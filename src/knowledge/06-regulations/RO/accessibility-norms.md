# Accessibility Norms (Romania)

## Overview
NP 051/2012 sets Romanian accessibility requirements for buildings, complementing the EU Accessibility Directive.

## Key Requirements

| Element | Requirement | NP 051 Reference |
|---------|------------|-----------------|
| Entrance door width | ≥ 900mm | Clear opening |
| Interior door width | ≥ 800mm | ≥ 900mm for wheelchair |
| Corridor width | ≥ 1200mm | Main circulation |
| Ramp gradient | ≤ 1:12 (8.3%) | Max 6m run, then 1500mm landing |
| Ramp width | ≥ 1000mm | With handrails |
| Threshold max | 20mm | Bevelled edge |
| Wheelchair turning | 1500mm ∅ | In key rooms and corridor ends |
| Lift (if provided) | 1100 × 1400mm min | Internal dimensions |
| Toilet height | 460–480mm | Accessible WC |
| Grab bars | 800–900mm from floor | Beside toilet and shower |
| Switch height | 800–1200mm | From finished floor |
| Socket height | 400–800mm | Accessible range |

## When Required
- **Public buildings**: Always required
- **Multi-family residential**: Common areas must be accessible
- **Single-family homes**: Recommended but not mandatory (unless publicly funded)
- **ModulCA approach**: Offer as standard option — future-proofing for aging in place

## ModulCA Application
The 3×3m module grid supports accessibility: single modules provide adequate turning circles when doors are positioned correctly. Two-module bathrooms easily accommodate all NP 051 requirements. Level thresholds between modules are standard.
