# Seismic Design (P100-1/2013)

## Overview
Romania's seismic code requirements relevant to modular construction.

## Seismic Zones

| Zone / Parameter | Value | Notes |
|-----------------|-------|-------|
| Peak ground accel. (ag) | 0.10–0.40g | Zone-dependent map |
| Bucharest zone | ag = 0.30g | Tc = 1.6s |
| Cluj zone | ag = 0.10g | Tc = 0.7s |
| Importance factor (γI) | 1.0 | Residential buildings |
| Behaviour factor (q) | 1.5–2.0 | Timber/light steel |
| Max. stories (light frame) | 2–3 | Without special analysis |

## Modular Construction Advantage
Modular buildings have inherently low mass compared to traditional masonry or reinforced concrete, resulting in lower seismic forces. The module-to-module bolt connections provide ductile behavior under seismic loading. Light timber/steel frame construction with proper connection design performs well in seismic zones.
