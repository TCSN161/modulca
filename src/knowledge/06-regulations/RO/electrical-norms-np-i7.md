# Electrical Installation Norms (NP I7)

## Overview
NP I7-2002 governs electrical installations in Romanian buildings. It covers circuit design, protection, grounding, and installation methods.

## Supply & Main Distribution

| Parameter | Value | Notes |
|-----------|-------|-------|
| Supply voltage | 230V / 400V, 50Hz | Single or three-phase |
| Typical residential supply | Single-phase, 32A | 7.4 kW capacity |
| Three-phase option | 3×32A | 22 kW — for heat pump + EV |
| Main breaker | 32A (single) / 3×32A (three) | At meter point |
| Consumer unit location | Entrance hallway | Accessible, protected |
| Earth system | TN-C-S or TT | Depends on supply network |

## Circuit Requirements

| Circuit | Cable | Protection | Notes |
|---------|-------|-----------|-------|
| Lighting | 1.5mm² | 10A MCB | Max 10 points per circuit |
| Socket outlets | 2.5mm² | 16A MCB + 30mA RCD | Ring or radial |
| Kitchen appliances | 2.5mm² | 16A MCB + 30mA RCD | Dedicated circuits for oven, dishwasher |
| Cooker circuit | 4.0mm² or 6.0mm² | 32A MCB | Dedicated |
| Bathroom (SELV zone) | 2.5mm² | 16A MCB + 30mA RCD | IP44+ for zones 1-2 |
| Outdoor circuits | 2.5mm² | 16A MCB + 30mA RCD | IP65 minimum |
| EV charger | 6.0mm² | 32A MCB + 30mA Type A RCD | Future-proofing |

## Outlet Minimum Counts

| Room | Min. Outlets | Recommended |
|------|-------------|-------------|
| Living room | 4 | 6–8 |
| Bedroom | 3 | 4–6 |
| Kitchen | 4 (above counter) | 6–8 + appliance circuits |
| Bathroom | 1 (outside zone 2) | 2 + shaver socket |
| Hallway | 1 | 2 |
| Home office | 4 | 6 + data |

## Safety Requirements

| Requirement | Standard | Notes |
|-------------|----------|-------|
| RCD protection | 30mA on all circuits | NP I7 requirement |
| Surge protection | Type 2 SPD | Recommended |
| Smoke detector circuit | Permanent live | Not switched |
| Cable routing | In conduit or behind lining | Concealed installation |
| Bathroom zones | 0, 1, 2, 3 per NP I7 | IP ratings per zone |

## ModulCA Application
All electrical wiring is factory-installed in conduit within the module walls. Each module has a sub-distribution point connecting to the main consumer unit via quick-connect jumpers. Pre-wired for EV charging (6mm² to garage/carport module).
