# Energy Standards — EPB/PEB (Belgium)

## Overview
Belgium's energy performance regulations are regional (EPB in Flanders, PEB in Wallonia/Brussels), but all implement the EU EPBD. Flanders is the strictest, with the E-peil (energy level) and S-peil (net energy demand) as key indicators. Since 2021, all new buildings must be "BEN" (bijna-energieneutrale) — nearly zero-energy.

## Flanders EPB Requirements (2025)

| Parameter | Value | Notes |
|-----------|-------|-------|
| E-peil (energy level) | ≤ E30 | Primary energy indicator (lower = better) |
| S-peil (net energy demand) | ≤ S28 | Replaces K-peil, measures building shell quality |
| U-value exterior wall | ≤ 0.24 W/m²K | Minimum requirement |
| U-value roof | ≤ 0.24 W/m²K | |
| U-value floor | ≤ 0.24 W/m²K | |
| U-value windows | ≤ 1.50 W/m²K | |
| Ventilation | System C or D | Mechanical extraction or balanced |
| Renewable energy | Integral part of E-peil | Lower E-peil with renewables |
| Overheating | Oververhittingsindicator ≤ 6500 Kh | Summer comfort |

## Wallonia PEB Requirements (2025)

| Parameter | Value | Notes |
|-----------|-------|-------|
| Ew (specific primary energy) | ≤ 45 kWh/m²·year | Equivalent to E-peil |
| U-value exterior wall | ≤ 0.24 W/m²K | Same as Flanders |
| U-value roof | ≤ 0.24 W/m²K | |
| Ventilation | System C or D | |
| Renewable energy share | ≥ 10 kWh/m²·year | Minimum renewable contribution |

## Brussels PEB Requirements (2025)

| Parameter | Value | Notes |
|-----------|-------|-------|
| PEB standard | Passive House equivalent | Brussels requires near-Passive House performance |
| Net heating demand | ≤ 15 kWh/m²·year | Passive House level |
| Primary energy | ≤ 45 kWh/m²·year | |
| Airtightness | n50 ≤ 0.6/h | Mandatory blower door test |

## Comparison Across Regions and Countries

| Parameter | Flanders | Wallonia | Brussels | Netherlands | Germany |
|-----------|----------|----------|----------|-------------|---------|
| Energy indicator | E30 | Ew 45 | Near-passive | BENG2 ≤ 50 | GEG ≤ 55 |
| Airtightness test | Recommended | Recommended | **Mandatory** | Calculated | Recommended |
| Strictness ranking | Very strict | Strict | **Most strict** | Very strict | Strict |
| Gas ban | Effectively (E-peil) | Not explicit | Yes (new areas) | Yes (2018) | Yes (2024) |
