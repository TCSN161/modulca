# Accessibility Requirements (Belgium)

## Overview

Accessibility in Belgian construction is regulated at the regional level, meaning Flanders, Wallonia, and Brussels each have distinct rules. The general principle across all regions is that publicly accessible buildings and multi-family housing must meet minimum accessibility standards. Single-family homes have fewer mandatory requirements, but the concept of "adaptable housing" (aanpasbare woning / logement adaptable) is increasingly promoted to future-proof dwellings for aging occupants.

## Regional Framework Comparison

| Aspect | Flanders | Wallonia | Brussels |
|--------|----------|----------|----------|
| Primary legislation | Besluit Vlaamse Regering 5/6/2009 (Toegankelijkheid) | CWATUP Art. 414-415 + AGW 2008 | RRU (Règlement Régional d'Urbanisme) Title IV |
| Scope | Public buildings + multi-family (>= 3 units) | Public buildings + multi-family | Public buildings + multi-family |
| Single-family | Not mandatory (recommended) | Not mandatory | Not mandatory |
| Enforcement body | Inter (Agentschap Toegankelijk Vlaanderen) | DGO4 (Urbanisme) | Urban.brussels |

## When Do Accessibility Rules Apply?

| Project Type | Flanders | Wallonia | Brussels |
|--------------|----------|----------|----------|
| New single-family home | No (voluntary) | No (voluntary) | No (voluntary) |
| New apartment building (>= 3 units) | Yes | Yes | Yes |
| Renovation of public building | Yes (if substantial) | Yes (if substantial) | Yes (if substantial) |
| Social housing (new) | Yes (stricter) | Yes (stricter) | Yes (stricter) |
| Care homes / assisted living | Yes (full compliance) | Yes (full compliance) | Yes (full compliance) |

## Key Dimensional Requirements

### Common Circulation Areas (Multi-Family)

| Element | Minimum Dimension | Standard |
|---------|------------------|----------|
| Entrance door width (clear) | 90 cm | All regions |
| Interior door width (clear) | 85 cm | All regions |
| Corridor width | 120 cm (150 cm for passing) | All regions |
| Lift cabin (min) | 110 cm x 140 cm | NBN EN 81-70 |
| Lift door width | 90 cm | NBN EN 81-70 |
| Ramp gradient (max) | 5% (6% if < 2 m length) | All regions |
| Level threshold (max) | 2 cm | All regions |

### Adaptable Dwelling Interior

| Element | Requirement |
|---------|------------|
| Living room / kitchen | Wheelchair turning circle 150 cm diameter |
| Bathroom | Minimum 170 cm x 220 cm (or equivalent for turning) |
| Bedroom (main) | Minimum 300 cm x 350 cm to allow bed access from three sides |
| Toilet | Minimum 170 cm x 200 cm; support rail provisions |
| Kitchen worktop | Adjustable height or knee space below for wheelchair access |
| Switches and outlets | Height: 40-120 cm above floor |
| Door handles | Lever type, 80-110 cm above floor |

## Flanders: Detailed Requirements

The Flemish accessibility decree (Besluit Vlaamse Regering 5 June 2009) is the most detailed of the three regions:

- **Applicability**: All new buildings requiring a permit with public function or >= 3 residential units
- **Access path**: From public road to entrance must be step-free, firm surface, min 150 cm wide
- **Parking**: 1 accessible parking space per 50 spaces (min 350 cm wide)
- **Lift**: Mandatory if building has > 1 floor and >= 3 units per floor
- **Signage**: Tactile floor indicators at stairs, lifts, and main routes in public buildings
- **Advisory body**: Inter (Agentschap Toegankelijk Vlaanderen) provides free advice and can issue a binding negative opinion

## Wallonia: Requirements

- **Legal basis**: CWATUP (Code Wallon de l'Aménagement du Territoire) and Arrêté du Gouvernement wallon of 2008
- **Scope**: Public buildings and multi-family residential (>= 3 units)
- **Key differences**: Less detailed than Flanders; relies more on the architect's professional judgment
- **Adaptable housing**: Encouraged but not strictly mandated for private residential

## Brussels: RRU Title IV

- **Legal basis**: RRU (Règlement Régional d'Urbanisme), Title IV on accessibility
- **Scope**: All new buildings and major renovations requiring a permit
- **Urban context**: Emphasis on street-level accessibility due to dense urban environment
- **Step-free access**: Mandatory for ground-floor commercial/public spaces
- **Historic buildings**: Exemptions possible for classified heritage buildings, but alternatives must be explored

## Adaptable Housing Concept

Belgium promotes "adaptable housing" — dwellings designed so they can be easily modified for wheelchair users without structural changes:

| Principle | Implementation |
|-----------|---------------|
| Wide doors | Install 93 cm door frames (gives 85+ cm clear passage) |
| Step-free shower | Floor-level shower tray or wet-room design |
| Reinforced walls | Plywood backing in bathroom walls for future grab rails |
| Open plan | Minimise interior load-bearing walls for future reconfiguration |
| Flexible bathroom | Space reservation for future wheelchair turning |

This concept is particularly suited to modular construction where these features can be designed into the standard module at minimal extra cost.

## Social Housing Requirements

Social housing in all three regions has stricter accessibility standards:

- **Flanders (VMSW)**: 100% of ground-floor units must be adaptable; lifts mandatory in all multi-storey buildings
- **Wallonia (SWL)**: Similar requirements with adaptability for aging population
- **Brussels (SLRB)**: Minimum percentage of fully accessible units in each project

## ModulCA Application

- **Standard module advantage**: Modular designs can incorporate accessibility from the outset without cost premium
- **Wide doors as default**: Using 93 cm frames across all modules meets accessibility thresholds
- **Level floors**: Factory-produced modules inherently provide level floors between rooms
- **Bathroom pods**: Pre-fabricated accessible bathrooms with reinforced walls and level-entry showers
- **Lift integration**: For stacked modules, lift shafts can be designed as a separate structural element
- **Future adaptability**: Modular non-load-bearing partitions allow easy reconfiguration

## Documentation for Permit

When accessibility rules apply, the permit application must include:

1. Accessibility note (toegankelijkheidsdossier / note d'accessibilité)
2. Floor plans showing wheelchair turning circles and clearances
3. Section drawings showing ramp gradients and lift positions
4. Material specifications for tactile indicators and signage (public buildings)
