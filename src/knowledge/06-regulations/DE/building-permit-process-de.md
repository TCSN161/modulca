# Building Permit Process (Germany)

## Overview
Germany's building regulations are primarily a state (Bundesland) responsibility. Each of the 16 states has its own Landesbauordnung (state building code), though they're based on the Musterbauordnung (MBO — Model Building Code). The permit process is thorough but well-structured.

## Permit Categories

| Category | Description | Permit Required? |
|----------|-------------|-----------------|
| Genehmigungsfreie Vorhaben | Small structures, carports, fences | No — but must comply with building code |
| Vereinfachtes Verfahren | Simple residential (Gebäudeklasse 1-3) | Simplified procedure |
| Baugenehmigung | Standard building permit | Full procedure |
| Bebauungsplan (B-Plan) area | Within zoning plan area | Must comply with B-Plan |
| Außenbereich (outside built area) | Building in countryside | Restricted — §35 BauGB |

## Permit Process Steps

| Step | Action | Who | Duration |
|------|--------|-----|----------|
| 1 | Bebauungsplan check | Owner/Architect | 1-2 weeks |
| 2 | Bauvoranfrage (preliminary inquiry) — optional | Architect → Bauamt | 4-8 weeks |
| 3 | Entwurfsplanung (design) | Architect (Bauvorlageberechtigter) | 4-8 weeks |
| 4 | Bauantrag (building application) | Architect → Bauamt | — |
| 5 | Prüfung (review by authorities) | Bauamt + Prüfstatiker | 4-12 weeks |
| 6 | Baugenehmigung (permit issued) | Bauamt | — |
| 7 | Baubeginnanzeige (start notification) | Owner → Bauamt | Before construction |
| 8 | Construction + inspections | Builder + Prüfingenieur | During construction |
| 9 | Fertigstellungsanzeige (completion) | Owner → Bauamt | After completion |

## Key Professionals Required

| Professional | German Title | Role |
|-------------|-------------|------|
| Architect | Bauvorlageberechtigter Architekt | Design + permit application (must be registered) |
| Structural engineer | Prüfstatiker / Tragwerksplaner | Structural calculations + verification |
| Energy consultant | Energieberater | GEG energy certificate |
| Surveyor | Vermessungsingenieur | Site survey, boundary check |
| Building supervisor | Bauleiter | Site supervision |

## Key Differences from Romania/Netherlands

| Aspect | Germany | Romania | Netherlands |
|--------|---------|---------|-------------|
| Permit authority | Bauamt (state level) | Primărie (local) | Municipality (Omgevingsloket) |
| Architect requirement | Bauvorlageberechtigter | Yes (authorized) | Not always required |
| Structural check | Independent Prüfstatiker | Yes (verificator) | Kwaliteitsborger (since 2024) |
| Process duration | 4-12 weeks | 3-6 months | 8-26 weeks |
| State variation | 16 different codes | National code | National code |

## ModulCA Application
- **Gebäudeklasse 1-3**: Single-family modular homes fall in GK 1-2 — simplified permit process
- **Architect required**: German-registered architect must sign the Bauantrag — partner needed per Bundesland
- **Prüfstatiker**: Independent structural check required — CLT/timber calculations must be verified
- **B-Plan compliance**: Module layout must fit within local Bebauungsplan requirements (height, footprint, setbacks)
- **Prefab advantage**: Factory-built modules can be pre-checked by Prüfingenieur — speeds site inspection
