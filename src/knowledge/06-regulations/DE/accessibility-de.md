# Accessibility Standards — DIN 18040 (Germany)

## Overview
Germany regulates barrier-free construction (barrierefreies Bauen) through DIN 18040, the primary accessibility standard for buildings. DIN 18040-1 covers publicly accessible buildings, DIN 18040-2 covers residential buildings. The MBO §50 mandates that buildings with more than 2 dwelling units must have barrier-free access to a proportion of units. State building codes (LBOs) define the exact percentages — typically one-third to all ground-floor units in multi-family buildings.

## DIN 18040 Parts

| Part | Scope | Key Application |
|------|-------|----------------|
| DIN 18040-1 (2010) | Public buildings (öffentlich zugängliche Gebäude) | Offices, shops, community buildings |
| DIN 18040-2 (2011) | Residential buildings (Wohnungen) | Apartments, single-family homes |
| DIN 18040-3 (2014) | Public outdoor spaces | Paths, parking, playgrounds |

## When Is Barrier-Free Design Required?

| Building Type | Requirement | Legal Basis |
|---------------|-------------|-------------|
| Multi-family (> 2 units) | Barrier-free access to a share of units | MBO §50, varies by LBO |
| Publicly funded housing | All units barrier-free | State funding conditions |
| Public buildings | Fully accessible | DIN 18040-1, MBO §50 |
| Single-family home | Not mandated (but recommended) | Optional, KfW incentives available |

State variation examples:
- **Bayern**: All ground-floor units in buildings with > 2 units must be barrier-free
- **NRW**: 20% of units (minimum) must be wheelchair-accessible in buildings with > 8 units
- **Berlin**: All units in new buildings must meet barrier-free standards (stricter than MBO)

## Key Dimensions — DIN 18040-2 (Residential)

### Movement Areas (Bewegungsflächen)

| Space | Barrier-Free (barrierefrei) | Wheelchair-Accessible (R-Standard) |
|-------|---------------------------|-------------------------------------|
| General movement area | 120 cm x 120 cm | 150 cm x 150 cm |
| Turning circle | 120 cm diameter | 150 cm diameter |
| Corridor/hallway width | ≥ 120 cm | ≥ 150 cm |
| Passing space in corridors | 180 cm width at intervals | 180 cm width at intervals |

### Doors

| Parameter | Barrier-Free | Wheelchair-Accessible (R) |
|-----------|-------------|---------------------------|
| Clear opening width | ≥ 80 cm | ≥ 90 cm |
| Clear opening height | ≥ 205 cm | ≥ 205 cm |
| Threshold height | ≤ 2 cm (beveled) | 0 cm (flush, level) |
| Movement area (push side) | 120 cm deep, ≥ 150 cm wide | 150 cm deep, ≥ 190 cm wide |
| Movement area (pull side) | 120 cm deep, ≥ 150 cm wide | 150 cm deep, ≥ 190 cm wide |
| Handle height | 85-105 cm | 85-105 cm |
| Operating force | ≤ 25 N (interior) | ≤ 25 N (interior) |

### Kitchens

| Parameter | Barrier-Free | Wheelchair-Accessible (R) |
|-----------|-------------|---------------------------|
| Movement area | 120 x 120 cm in front of each appliance | 150 x 150 cm in front of each appliance |
| Countertop height | 82-85 cm (or adjustable) | Adjustable or underride-capable |
| Under-counter clearance | Not required | ≥ 67 cm height, ≥ 30 cm depth |

### Bathrooms

| Parameter | Barrier-Free | Wheelchair-Accessible (R) |
|-----------|-------------|---------------------------|
| Movement area beside WC | 90 cm wide | 95 cm wide, both sides |
| Movement area in front of WC | 120 x 120 cm | 150 x 150 cm |
| WC seat height | 46-48 cm | 46-48 cm |
| Shower | Level-entry (bodengleich) | Level-entry, 150 x 150 cm min. |
| Washbasin clearance underneath | 55 cm height | 67 cm height |
| Grab bars | At WC and shower | At WC, shower, and bathtub |
| Mirror | Lower edge ≤ 100 cm | Lower edge ≤ 85 cm |

### Ramps

| Parameter | Requirement |
|-----------|-------------|
| Maximum gradient | 6% (1:16.7) |
| Maximum single ramp run | 6 m length |
| Ramp width | ≥ 120 cm (barrier-free), ≥ 150 cm (wheelchair) |
| Level landing between runs | ≥ 150 cm long |
| Handrails | Both sides, 85-90 cm height, extending 30 cm beyond ramp |
| Edge protection | ≥ 10 cm curb on both sides |

### Stairs

| Parameter | Requirement |
|-----------|-------------|
| Stair width | ≥ 100 cm (residential), ≥ 150 cm (public) |
| Handrails | Both sides, 85-90 cm height |
| Handrail extension | 30 cm beyond top and bottom |
| Step marking | Contrasting strips on nosings (visually impaired) |
| Rise (Steigung) | 15-17 cm recommended |
| Tread depth (Auftritt) | ≥ 26 cm |

## Elevators (Aufzüge)

When an elevator is required (typically buildings with > 3 stories or barrier-free requirement):

| Parameter | Barrier-Free | Wheelchair-Accessible (R) |
|-----------|-------------|---------------------------|
| Cabin minimum (width x depth) | 100 cm x 125 cm | 110 cm x 140 cm |
| Door clear width | ≥ 80 cm | ≥ 90 cm |
| Control panel height | 85-115 cm | 85-115 cm |
| Tactile/Braille buttons | Required | Required |
| Movement area at each door | 150 x 150 cm | 150 x 150 cm |

| Program | Benefit | For |

## ModulCA Application
- **Modular flexibility**: Module dimensions can be designed to accommodate 150 cm turning circles — plan at module design stage
- **Door widths**: Standard ModulCA door openings should be ≥ 90 cm to meet the wheelchair-accessible R-standard by default
- **Level-entry showers**: Factory-installed level-entry showers are easy to produce — barrier-free bathrooms at no extra cost
- **Ramp-ready entrances**: ModulCA entrance modules can include integrated ramps or level thresholds (max 2 cm) as standard
- **KfW double benefit**: Combining KfW energy subsidies (261) with accessibility funding (159) maximizes financial advantage
- **Aging-in-place selling point**: Barrier-free modular homes appeal to Germany's aging demographic — strong market argument
