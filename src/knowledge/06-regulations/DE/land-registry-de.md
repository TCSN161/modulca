# Land Registry and Zoning (Germany)

## Overview
Germany has one of Europe's most reliable land registration systems. The Grundbuch (land register) provides legally binding ownership records, while the Kataster (cadastral register) maps the physical boundaries of every parcel. Before building, the local zoning framework — Flächennutzungsplan (land use plan) and Bebauungsplan (binding development plan) — determines what can be built on a specific plot. Understanding these systems is essential before purchasing land for modular construction.

## The Grundbuch (Land Register)

The Grundbuch is maintained by the local Amtsgericht (district court) and provides legal certainty about property ownership and encumbrances.

### Grundbuch Structure

| Section (Abteilung) | Content | Relevance for Building |
|---------------------|---------|----------------------|
| Bestandsverzeichnis | Plot description (Gemarkung, Flur, Flurstück number) | Identifies the exact parcel |
| Abteilung I | Owner(s) | Confirms who can apply for a building permit |
| Abteilung II | Encumbrances and restrictions | Easements (Dienstbarkeiten), building restrictions (Baulasten), rights of way |
| Abteilung III | Mortgages and charges | Grundschuld, Hypothek — relevant for financing |

### Key Encumbrances to Check (Abteilung II)

| Encumbrance | German Term | Impact on Building |
|-------------|-------------|-------------------|
| Right of way | Wegerecht | Third party has access across the plot |
| Building easement | Baulast | May restrict building height, setback, or use |
| Conservation easement | Denkmalschutz | Heritage restrictions on what can be built |
| Utility easement | Leitungsrecht | Underground pipes/cables — affects foundation placement |
| Pre-emptive purchase right | Vorkaufsrecht | Municipality may have first right to purchase |
| Reallast | Recurring obligation | Ongoing duties tied to the land |

### How to Access the Grundbuch
- **Who can view**: Owner, authorized parties (notary, bank, buyer with legitimate interest)
- **Where**: Grundbuchamt at the local Amtsgericht
- **Cost**: Grundbuchauszug (extract) costs approximately 10-20 EUR
- **Online**: Some states offer electronic access via grundbuch-portal.de (with authentication)

## The Kataster (Cadastral Register)

The Kataster is maintained by the Katasteramt (surveying office) or Vermessungsamt and maps physical plot boundaries.

| Document | Content | Use |
|----------|---------|-----|
| Flurkarte (cadastral map) | Plot boundaries, building footprints | Verify plot shape and neighbors |
| Liegenschaftsbuch | Area, land use category | Cross-reference with Grundbuch |
| Amtlicher Lageplan | Detailed site plan with heights, boundaries | Required for Bauantrag (building application) |

For building permits, a Vermessungsingenieur (licensed surveyor) prepares the Amtlicher Lageplan based on Kataster data plus an on-site survey. This typically costs 1,500-3,000 EUR.

## Zoning Framework (Bauleitplanung)

German zoning operates on two levels under the Baugesetzbuch (BauGB):

### 1. Flächennutzungsplan (FNP) — Preparatory Land Use Plan

| Designation | Description | Building Potential |
|-------------|-------------|-------------------|
| Wohnbaufläche (W) | Residential area | High — residential permitted |
| Gemischte Baufläche (M) | Mixed use | Residential + commercial |
| Gewerbliche Baufläche (G) | Commercial/industrial | No residential (usually) |
| Sonderbaufläche (S) | Special use | Depends on designation |
| Grünfläche | Green/park land | No building |
| Landwirtschaftsfläche | Agricultural | Very restricted (§35 BauGB) |

The FNP is not directly binding on property owners but guides the creation of Bebauungspläne.

### 2. Bebauungsplan (B-Plan) — Binding Development Plan

The B-Plan is the legally binding zoning document. It defines exactly what can be built.

| Parameter | Symbol/Code | Example |
|-----------|------------|---------|
| Area type | WA, WR, MI, etc. | WA = Allgemeines Wohngebiet (general residential) |
| GRZ (Grundflächenzahl) | 0.0-1.0 | 0.4 = max. 40% of plot can be built on |
| GFZ (Geschossflächenzahl) | 0.0-3.0+ | 0.8 = total floor area ≤ 80% of plot area |
| Number of stories (Vollgeschosse) | Z (Zahl) | II = max. 2 full stories |
| Building height | Firsthöhe / Traufhöhe | Max. ridge height / eave height in meters |
| Roof form | SD, FD, PD, WD | SD = Satteldach (gable), FD = Flachdach (flat) |
| Roof pitch | ° | e.g., 25-45 degrees |
| Building line (Baulinie) | Red line on plan | Building facade MUST align with this line |
| Building limit (Baugrenze) | Blue line on plan | Building must NOT exceed this boundary |
| Setback (Abstandsfläche) | Per LBO | Typically 0.4H (H = building height), min. 3m |

### Area Types (BauNVO)

| Code | German Name | Description |
|------|------------|-------------|
| WS | Kleinsiedlungsgebiet | Small settlement (allotments) |
| WR | Reines Wohngebiet | Pure residential — only housing |
| WA | Allgemeines Wohngebiet | General residential — housing + small shops/offices |
| MI | Mischgebiet | Mixed use — equal residential + commercial |
| MK | Kerngebiet | City center — dense mixed use |
| GE | Gewerbegebiet | Commercial/light industrial |
| GI | Industriegebiet | Heavy industrial |

## Building Rights Without a B-Plan

Not all areas have a Bebauungsplan. In unbeplanter Innenbereich (built-up areas without a B-Plan), §34 BauGB applies:

| Criterion | Requirement |
|-----------|-------------|
| Type of use | Must fit the surrounding area (Eigenart der näheren Umgebung) |
| Building dimensions | Must match neighboring buildings in volume and height |
| Plot coverage | In keeping with surroundings |
| Infrastructure | Must be served by roads and utilities |

In the Außenbereich (outside built-up areas), §35 BauGB applies very restrictively — generally only agricultural buildings, wind turbines, and specific privileged uses are permitted.

## Due Diligence Checklist for Plot Purchase

| Step | Action | Source |
|------|--------|--------|
| 1 | Obtain Grundbuchauszug | Grundbuchamt (Amtsgericht) |
| 2 | Check Abteilung II for encumbrances | Grundbuchauszug |
| 3 | Obtain Flurkarte | Katasteramt |
| 4 | Check Bebauungsplan (if exists) | Municipality (Bauamt), often online |
| 5 | If no B-Plan: request Bauvorbescheid (preliminary building decision) | Bauamt (§34 assessment) |
| 6 | Check Altlastenkataster (contamination register) | Umweltamt |
| 7 | Check flood zone maps (Hochwasserschutz) | Wasserbehörde |
| 8 | Check monument protection (Denkmalschutz) | Denkmalamt |
| 9 | Verify utility connections available | Local Stadtwerke |
| 10 | Commission Bodengutachten (soil report) | Geotechnical engineer |

## ModulCA Application
- **B-Plan compliance check**: ModulCA's configurator can pre-validate designs against GRZ, GFZ, height, and roof form constraints from the local B-Plan
- **Module footprint optimization**: GRZ limits mean ModulCA layouts must optimize coverage — configuration tool should flag when limits are exceeded
- **Flat roof compatibility**: Many newer B-Plans allow or require flat roofs (Flachdach) — ideal for modular construction with less complex roof modules
- **Abstandsflächen**: Modular layouts must respect setback requirements — typically 3m to boundary or 0.4x building height
- **Bauvorbescheid first**: For plots without a B-Plan, always recommend clients obtain a preliminary building decision before committing to purchase or module configuration
