# Accessibility Standards (Netherlands)

## Overview
Dutch accessibility requirements for new dwellings are defined in BBL Chapter 4. The Netherlands has relatively progressive accessibility standards, especially for social housing.

## Key Requirements (New Woning)

| Element | Requirement | Notes |
|---------|------------|-------|
| Entrance level | Accessible entrance required | Ground floor or lift access |
| Door width (entrance) | ≥ 850mm clear | All entrance doors |
| Door width (internal) | ≥ 750mm clear | Habitable rooms |
| Corridor width | ≥ 1.20m | Main circulation |
| Threshold max | ≤ 20mm | All transitions |
| Toilet on entrance level | Required | Accessible WC |
| Living area on entrance level | Required | Verblijfsruimte accessible |

## Aanpasbaar Bouwen (Adaptable Construction)
Dutch regulations encourage "aanpasbaar bouwen" — buildings designed so they can be easily adapted for wheelchair use later without structural changes.

| Feature | Standard | Adaptable |
|---------|----------|-----------|
| Door width | 750mm internal | Pre-framed for 850mm |
| Bathroom | Standard fixtures | Reinforced walls for grab bars |
| Shower | Raised tray acceptable | Floor drain prepared |
| Ground floor | Must function as complete dwelling | Sleeping possible on ground floor |
| Turning space | Not required by default | Structure allows 1500mm ∅ circles |

## Social Housing (Sociale Huurwoning)
Social housing has stricter accessibility requirements:

| Element | Requirement | Notes |
|---------|------------|-------|
| All doors | ≥ 850mm clear | Including internal |
| Turning circles | 1500mm ∅ | In kitchen, bathroom, bedroom |
| Kitchen counter | 850mm height | Lower section for wheelchair |
| Bathroom | Wheelchair-accessible | Full accessible bathroom |
| Lift (if > 3 floors) | Required | Min 1100 × 1400mm internal |

## Woonkeur (Housing Quality Mark)
Woonkeur is a voluntary quality certification that includes enhanced accessibility:

| Level | Description | Target |
|-------|-------------|--------|
| Basis | Meets BBL minimum | All new housing |
| Plus | Enhanced accessibility | Elderly-suitable housing |
| Top | Full wheelchair accessibility | Wheelchair-adapted housing |

## ModulCA Application
The 3×3m module system supports Dutch accessibility well:
- Module width (3m) exceeds corridor minimum (1.20m)
- 850mm doors are standard option
- Ground floor can accommodate toilet + living + sleeping (aanpasbaar)
- Module walls pre-reinforced for future grab bar installation
- Level thresholds between modules standard
