# Fire Safety Regulations (Netherlands)

## Overview
Dutch fire safety (brandveiligheid) is regulated through BBL Chapter 2 and detailed in NEN standards. Requirements are generally stricter than Romanian equivalents, especially for multi-family housing.

## Fire Resistance Requirements

| Element | Single-family | Multi-family | Notes |
|---------|--------------|-------------|-------|
| Load-bearing structure | WBDBO 60 min | WBDBO 60 min | Between fire compartments |
| Floor between dwellings | — | WBDBO 60 min | Between units |
| Separating wall (units) | — | WBDBO 60 min | Party wall |
| Roof structure | WBDBO 30 min | WBDBO 60 min | If accessible |
| Facade | See below | See below | Special requirements |

*WBDBO = Weerstand tegen branddoorslag en brandoverslag (resistance to fire passage)*

## Facade Requirements
Post-Grenfell, the Netherlands tightened facade fire safety:
| Building Height | Requirement | Notes |
|----------------|------------|-------|
| ≤ 13m (low-rise) | No specific cladding class | Common for modular homes |
| > 13m | Cladding ≥ Class B | Euroclass, limited combustibility |
| > 50m | Non-combustible cladding | Class A2 or better |

## Detection & Escape

| Requirement | Standard | Notes |
|-------------|----------|-------|
| Smoke detectors | Required in each floor level | Since BBL 2003, interconnected |
| CO detectors | If fuel-burning appliance | Mandatory where applicable |
| Escape route length | ≤ 30m to exit | Single direction |
| Exit door width | ≥ 850mm | Clear opening |
| Self-closing doors | Between garage and dwelling | Fire doors |
| Emergency lighting | Multi-family common areas | Not required for single-family |

## Fire Compartment Size
| Building Type | Max. Compartment | Notes |
|--------------|-----------------|-------|
| Single dwelling | 1 dwelling = 1 compartment | No internal fire walls needed |
| Multi-family | Each dwelling = 1 compartment | WBDBO 60 between units |
| Max. compartment area | 1,000 m² residential | Rarely relevant for homes |

## ModulCA Application
- Single-family modular homes: REI 60 module construction exceeds Dutch minimum
- Multi-family: Module-to-module fire compartmentation must achieve WBDBO 60
- Factory-installed smoke detectors (interconnected across modules)
- No gas connection (aardgasvrij) eliminates combustion-related fire risk
- CLT fire performance: predictable charring rate provides inherent fire resistance
