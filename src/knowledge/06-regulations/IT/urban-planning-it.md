# Urban Planning & Zoning — Italy

## Overview

Italian urban planning is governed by the **Testo Unico dell'Edilizia** (DPR 380/2001) and regional planning laws. Each municipality has a master plan (PRG, PGT, or PRGC depending on the region) that defines land use and building rules.

## Planning Instruments

### Municipal Level
| Instrument | Region | Purpose |
|-----------|--------|---------|
| **PRG** (Piano Regolatore Generale) | Most regions | Master development plan |
| **PGT** (Piano di Governo del Territorio) | Lombardia | Three-part plan system |
| **PRGC** (Piano Regolatore Generale Comunale) | Piemonte | Municipal master plan |
| **PAT** (Piano di Assetto del Territorio) | Veneto | Structural + operational plan |

### Zoning Categories
| Zone | Name | Description |
|------|------|------------|
| **A** | Centro storico | Historic center — heavy restrictions |
| **B** | Completamento | Built-up residential — infill allowed |
| **C** | Espansione | Expansion zones — new development |
| **D** | Produttiva | Industrial/commercial |
| **E** | Agricola | Agricultural — limited building |
| **F** | Attrezzature | Public facilities, parks |

### Key Building Parameters (NTA — Norme Tecniche di Attuazione)
| Parameter | Italian Term | Typical Range |
|-----------|-------------|--------------|
| Plot ratio (floor area) | Indice di edificabilità fondiaria (If) | 0.30–1.50 m³/m² |
| Ground coverage | Rapporto di copertura (Rc) | 30–50% |
| Max height | Altezza massima (Hmax) | 7.50–10.50m |
| Max floors | Numero piani | 2–3 (Zone B/C residential) |
| Distance from boundary | Distanza dai confini (Dc) | 5m minimum |
| Distance between buildings | Distanza tra edifici (De) | 10m minimum |
| Green area ratio | Superficie permeabile (Sp) | 30–50% of plot |

## How to Check Buildability

1. **Certificato di Destinazione Urbanistica (CDU)** — from Comune (valid 1 year)
2. Check the **NTA** (technical building rules) of the PRG/PGT
3. Verify zone classification and building indices
4. Check for vincoli (constraints): landscape, hydrogeological, archaeological
5. **Sportello Unico Edilizia (SUE)** — municipal one-stop building office

## ModulCA Application

Italian urban planning for modular construction:
- **Zone B/C**: Primary target for ModulCA (existing residential + expansion)
- **Zone E (agricola)**: Limited — some regions allow 0.03 m³/m² for agricultural use
- **Indice fondiario**: Determines maximum building volume → module count
- **10m rule**: Minimum distance between buildings — important for plot analysis
- Example: 600m² plot, Zone C, If=0.50 m³/m², Hmax=7.50m → 300m³ volume → ~33 modules (assuming 2.7m height)
