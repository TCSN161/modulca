# Accessibility Standards — Italy

## Overview

Italian accessibility regulations focus on eliminating **barriere architettoniche** (architectural barriers). The framework is based on **DM 236/1989** (technical requirements) and **Legge 13/1989** (mandatory compliance for new buildings).

## Three Levels of Accessibility

Italy uniquely defines three progressive levels:

| Level | Italian | Description |
|-------|---------|------------|
| **Accessibilità** | Full access | Independent use by wheelchair users |
| **Visitabilità** | Visitability | Guest access to living room, one WC, kitchen |
| **Adattabilità** | Adaptability | Can be converted to accessible with minor works |

### Requirements by Building Type
- **New apartments**: Must be **adattabile** (all) + **visitabile** (ground floor units)
- **Public housing**: Must be **accessibile** (5% of units)
- **Single-family homes**: Must be **adattabile** minimum

## Key Dimensions (DM 236/1989)

| Element | Minimum |
|---------|---------|
| Corridor width | 1.00m (1.50m for wheelchair passing) |
| Door clear opening | 0.80m |
| Wheelchair turning space | 1.50m × 1.50m |
| Ramp gradient | ≤8% (length ≤10m) |
| Ramp width | 1.50m minimum |
| Elevator cabin | 1.30m × 0.95m minimum |
| Threshold height | ≤25mm |

### Accessible Bathroom (DM 236/1989)
- Minimum area: ~4.5m² with 1.50m turning circle
- WC: Side transfer space ≥1.00m
- Grab bars: L-shaped, horizontal at 800mm from floor
- Washbasin: Height 800mm, knee clearance below
- Shower: Level-access, minimum 900×900mm, folding seat
