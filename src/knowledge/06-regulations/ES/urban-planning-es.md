# Urban Planning & Zoning — Spain

## Overview

Spanish urban planning is a **three-level system**: national framework (Ley del Suelo), autonomous community legislation, and **municipal plans (PGOU — Plan General de Ordenación Urbana)**. Each municipality defines its own building rules within the regional framework.

## Land Classification

### Three Categories (Clasificación del Suelo)

| Type | Spanish | Buildable? | Description |
|------|---------|-----------|-------------|
| **Urban** | Suelo Urbano | ✅ Yes | Fully serviced, building permitted |
| **Developable** | Suelo Urbanizable | ⚠️ With plan | Designated for future development, needs partial plan |
| **Rural/Protected** | Suelo No Urbanizable | ❌ No (normally) | Agricultural, forest, protected — very limited building |

### Key Municipal Parameters

| Parameter | Spanish Term | Typical Range |
|-----------|-------------|--------------|
| Plot ratio | Edificabilidad | 0.3–1.5 m²/m² |
| Ground coverage | Ocupación máxima | 30–60% |
| Max height | Altura máxima | 7–9m (2 floors residential) |
| Front setback | Retranqueo frontal | 3–5m |
| Side setback | Retranqueo lateral | 2–3m |
| Rear setback | Retranqueo posterior | 3–5m |
| Min plot size | Parcela mínima | 200–500m² (urban) |
| Max floors | Número de plantas | 2–3 (residential zones) |

## Planning Documents

1. **PGOU / POUM** — General plan (whole municipality)
2. **Normas Subsidiarias** — Simplified plan for small municipalities
3. **Plan Parcial** — Detailed plan for specific development areas
4. **Estudio de Detalle** — Fine-grain design for specific plots

## How to Check Buildability

1. Visit municipal **urbanismo** department (concejalía de urbanismo)
2. Request **cédula urbanística** — official document stating what you can build
3. Check PGOU online (many municipalities publish digital maps)
4. Verify **servicios urbanos**: water, sewage, electricity, road access

## ModulCA Application

For ModulCA's land analysis:
- **Edificabilidad** determines maximum number of modules
- **Ocupación** determines ground floor footprint
- **Retranqueos** define the buildable zone within the plot
- Example: 500m² plot, 40% ocupación, 0.6 edificabilidad = max 200m² ground floor, 300m² total → ~33 modules
