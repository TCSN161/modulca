# Fire Safety Regulations — Spain

## Overview

Spanish fire safety is governed by **CTE DB-SI** (Documento Básico — Seguridad en caso de Incendio), part of the Código Técnico de la Edificación. It defines requirements for fire resistance, evacuation, fire spread prevention, and firefighter access.

## Key Requirements for Residential Buildings

### Fire Resistance (DB-SI 6)
| Element | Requirement |
|---------|------------|
| Load-bearing structure | R 30 (single dwelling) to R 60 (apartment blocks) |
| Party walls | EI 60 minimum |
| Floor between dwellings | REI 60 |
| Roof structure | R 30 |
| Facade elements | Class B-s3,d0 minimum above 18m height |

### Evacuation (DB-SI 3)
- **Maximum travel distance** to exit: 25m (single exit) / 50m (two exits)
- **Minimum corridor width**: 1.00m
- **Minimum door width**: 0.80m (≤50 persons)
- **Staircase width**: 1.00m minimum for residential

### Fire Detection
- Residential <5,000m²: Smoke detectors in each dwelling required
- Fire extinguisher: One per 15m of travel distance, at least one per floor

## ModulCA Application

ModulCA's CLT timber modules require particular attention:
- **CLT fire resistance**: Charring rate 0.65mm/min — design for sacrificial layer
- **Module connections**: Fire stops at every module junction
- **Escape routes**: Max 1 module distance to corridor/exit in layouts >4 modules
