# Natural Light in Biophilic Design

## Overview
Natural light is the single most important biophilic design element. It regulates circadian rhythms, improves mood, reduces energy consumption, and connects occupants to the time of day and seasons. Dynamic daylight — light that changes throughout the day — is far more beneficial than static artificial light.

## Daylight Design Strategies

| Strategy | Description | Biophilic Pattern |
|----------|-------------|-------------------|
| Side windows | Standard windows in walls | Visual connection, dynamic light |
| Skylights | Roof-mounted glazing | Diffuse light, sky connection |
| Light wells | Vertical shafts bringing light deep | Mystery, dynamic light |
| Clerestory windows | High windows above eye level | Diffuse light, privacy |
| Light shelves | Horizontal reflectors above windows | Deeper daylight penetration |
| Translucent walls | Diffused light through panels | Soft, even illumination |

## Daylight Factor Targets

| Room | Minimum DF | Recommended DF | Notes |
|------|-----------|----------------|-------|
| Living room | 1.5% | 2.0-3.0% | South/east orientation |
| Bedroom | 1.0% | 1.5-2.0% | Controllable (blackout) |
| Kitchen | 1.5% | 2.0-3.0% | Task lighting supplement |
| Bathroom | 0.5% | 1.0-1.5% | Privacy glass option |
| Home office | 2.0% | 3.0-5.0% | Glare control needed |

## Circadian Lighting Design

| Time | Light Quality | Design Response |
|------|--------------|-----------------|
| Morning (6-9h) | Warm, increasing intensity | East-facing windows in bedroom/kitchen |
| Midday (9-15h) | Bright, blue-rich | South-facing windows, skylights |
| Afternoon (15-18h) | Warm, decreasing | West windows (with shading) |
| Evening (18-22h) | Warm, dim | Artificial: 2700K, dimmable |

## Window-to-Floor Ratios

| Standard | Minimum | Recommended |
|----------|---------|-------------|
| Romanian building code | 1:8 (12.5%) | 1:6 (16.7%) |
| Dutch BBL | 0.5m² per room | Varies by room |
| Passive House | Balance with thermal | Triple glazing allows more |
| WELL Building | — | 75% of workstations within 7.5m of window |

## ModulCA Application
- **3m module width**: Allows full-width window walls (2.40m clear glazing possible)
- **Dual-aspect modules**: Corner modules get windows on two walls — cross-light
- **Skylight modules**: Flat roof allows roof windows or tubes per module
- **Module orientation**: Place living modules south, bedrooms east for circadian benefit
- **Deep plan solution**: Central modules use skylight tubes or clerestory glazing
- **Seasonal response**: Deciduous trees on south side — sun in winter, shade in summer
