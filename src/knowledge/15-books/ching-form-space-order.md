# Francis Ching — Architecture: Form, Space, and Order

## Overview

**Francis D.K. Ching** is the master of architectural drawing and visual communication. *Form, Space, and Order* (4th edition, 2014) is arguably the best introduction to architectural design principles ever written. Its hand-drawn illustrations explain spatial concepts better than any photograph.

Every architecture student worldwide studies this book in their first year.

## Core Concepts

### Primary Elements
- **Point** → defines position in space
- **Line** → describes the path of a point; creates edges, boundaries
- **Plane** → defines surface; floor, wall, ceiling
- **Volume** → enclosed space; the room itself

### Form Properties
| Property | Description | ModulCA Example |
|----------|-------------|-----------------|
| **Shape** | Outline/silhouette | 3×3m square module base |
| **Size** | Physical dimensions | Module combinations (1→12 modules) |
| **Color** | Surface appearance | Material palette choices |
| **Texture** | Surface quality | Wood grain, concrete, plaster |
| **Position** | Location relative to context | Module placement on land |
| **Orientation** | Direction it faces | Solar orientation, view direction |

### Spatial Organization Patterns

Ching identifies 5 fundamental ways to organize spaces:

1. **Centralized** — Spaces around a dominant center (courtyard house)
2. **Linear** — Spaces in a row (ModulCA row layout)
3. **Radial** — Paths extending from a center
4. **Clustered** — Grouped by proximity (ModulCA cluster layout)
5. **Grid** — Modular repetition (★ ModulCA's core DNA)

ModulCA's module system is inherently a **grid** organization that enables **linear** and **clustered** configurations.

## Why It Matters

Ching teaches you to *see* architecture — to understand why some spaces feel right and others don't. The principles of proportion, hierarchy, rhythm, and datum that he explains are embedded in ModulCA's layout algorithms:

- **Hierarchy**: Living spaces are larger (2-3 modules) vs. utility (1 module)
- **Rhythm**: Repeating 3m structural grid creates visual regularity
- **Datum**: A shared wall or corridor line that aligns rooms
- **Proportion**: Room ratios that feel balanced (guided by Neufert + Ching)
