# Minimum Room Sizes (Romanian / EU)

## Overview
Legal and ergonomic minimums for habitable rooms per Romanian building code and Neufert guidelines.

## Standards

| Room | Minimum Area | Notes |
|------|-------------|-------|
| Single bedroom | ≥ 8 m² | Min. width 2.20m (RO P100) |
| Double bedroom | ≥ 12 m² | Min. width 2.60m |
| Living room | ≥ 14 m² | Min. width 3.00m |
| Kitchen | ≥ 5 m² | ≥ 6 m² if dining included |
| Bathroom | ≥ 2.5 m² | ≥ 3.5 m² with bathtub |
| WC (separate) | ≥ 1.2 m² | Min. 0.90 × 1.30m |
| Hallway width | ≥ 1.10m | ≥ 1.20m if wheelchair accessible |
| Home office | ≥ 6 m² | Neufert recommendation |
| Storage / utility | ≥ 2 m² | Per dwelling unit |

## ModulCA Application
In the 3×3m modular system (9m² gross, ~7m² usable per module):
- **Single bedroom**: One module fits comfortably with bed, nightstand, and wardrobe
- **Double bedroom**: Two adjacent modules with shared wall removed (14m² usable)
- **Living room**: Two modules minimum (14m²), three in L-shape for 21m²
- **Kitchen**: One module for L-shape kitchen, two for kitchen-dining with island
- **Bathroom**: One module fits full bathroom with shower, vanity, and toilet
