# Universal Design Principles

## Overview
Making modular homes accessible for all abilities — doorways, ramps, and fixtures.

## Standards

| Element | Requirement | Notes |
|---------|------------|-------|
| Door clear width | ≥ 900mm | ≥ 850mm absolute minimum |
| Corridor width | ≥ 1200mm | Wheelchair passage |
| Ramp gradient | ≤ 1:12 (8.3%) | Max 6m run, then landing |
| Threshold height | ≤ 20mm | Bevelled edge |
| Turning circle | 1500mm ∅ | In kitchen, bathroom, hallway |
| Grab bar height | 800–900mm | Beside toilet and shower |
| Toilet seat height | 460–480mm | Raised / comfort height |
| Switch height | 900–1100mm | From finished floor |
| Socket height | 400–1000mm | Accessible range |
| Level entry | ±0mm | Zero threshold at entrance |

## ModulCA Application
The 3×3m module system supports accessible design with 1500mm turning circles achievable in combined-module spaces. Level thresholds between modules are standard, and all doorways can be specified at 900mm clear width.
