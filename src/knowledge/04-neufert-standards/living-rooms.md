# Living Room Design Standards

## Overview
Neufert standards for living room design, including furniture placement, viewing distances, and circulation.

## Area Standards

| Configuration | Area | Notes |
|--------------|------|-------|
| Small living room | ≥ 16m² | Minimum for 2-person household |
| Family living room | 20–30m² | Comfortable for 4+ people |
| Open-plan living/dining | 30–40m² | Combined function |
| Min. width | 3.00m | Romanian building code |

## Furniture & Viewing Distances

| Parameter | Value | Notes |
|-----------|-------|-------|
| Sofa-to-TV distance | 2.5m for 55" | 3.0m for 65" screen |
| Coffee table clearance | 0.45m from sofa | Leg room |
| Circulation passage | min 0.90m | Between furniture |
| Bookshelf depth | 0.25–0.30m | Standard shelving |
| Fireplace clearance | ≥ 1.00m | Combustible materials |

## Lighting

| Type | Level | Notes |
|------|-------|-------|
| General ambient | 150–200 lux | Overhead or indirect |
| Reading task light | 300–500 lux | Beside chair/sofa |
| TV viewing ambient | 50–100 lux | Reduce glare |
| Natural light | min 1/8 floor area | Window requirement |

## ModulCA Application
- **2 modules** (shared wall removed): 14m² usable — compact living for 2 people
- **3 modules** (L-shape): 21m² usable — comfortable family living
- **4 modules** (open plan): 28m² — generous living/dining combined
- South/south-east orientation recommended for natural light and solar gain
