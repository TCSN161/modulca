# Ceiling Heights

## Overview
Minimum and recommended clear heights for modular construction.

## Standards

| Space | Height | Notes |
|-------|--------|-------|
| Habitable room min. | 2.50m | Romanian building code |
| Recommended residential | 2.60–2.70m | Neufert optimal comfort |
| ModulCA standard | 2.70m | Clear interior height |
| Bathroom / kitchen | ≥ 2.40m | Allowed reduction in some codes |
| Mezzanine / loft | ≥ 2.20m | At lowest point |
| Corridor / hallway | ≥ 2.40m | Can be lower than main rooms |

## ModulCA Application
ModulCA modules use a standard 2.70m clear interior height, exceeding the Romanian minimum of 2.50m. This provides a more spacious feel and accommodates suspended ceilings or MEP routing while still meeting minimum clearances.
