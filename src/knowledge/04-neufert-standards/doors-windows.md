# Doors & Windows

## Overview
Neufert standard dimensions for doors and windows in residential buildings.

## Door Standards

| Type | Width | Height | Notes |
|------|-------|--------|-------|
| Interior door | 0.80m | 2.01m | Standard residential |
| Main entrance | ≥ 1.00m | 2.10m | Minimum |
| Accessible door | ≥ 0.90m clear | 2.10m | Clear opening width |
| Door swing clearance | 0.80m radius | — | Free space for swing |
| Corridor doors | Same as corridor min | — | Must match corridor width |

## Window Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Natural light | min 1/8 of floor area | Window glazed area |
| Ventilation | min 1/20 of floor area | Openable area |
| Sill height | 0.85–0.90m | Standard residential |
| Safety glazing | Below 0.80m sill | Required for low windows |

## Romanian Specifics
- Main entrance: min 0.90m wide
- Fire doors: REI 30 rating minimum
- All external doors must be insulated (U ≤ 1.80 W/m²K)

## ModulCA Application
ModulCA modules use standard 0.80m interior doors and 0.90m entrance doors. Windows are pre-installed in the factory, positioned to maximize natural light (minimum 1/8 floor area ratio per room).
