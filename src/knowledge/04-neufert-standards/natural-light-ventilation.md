# Natural Light & Ventilation Standards

## Overview
Neufert and EU standards for daylighting and natural ventilation in residential buildings.

## Daylighting Requirements

| Parameter | Value | Notes |
|-----------|-------|-------|
| Window-to-floor ratio | ≥ 1/8 (12.5%) | Glazed area to floor area |
| Daylight factor | ≥ 2% | At room centre, overcast sky |
| Min window head height | 2.10m | Higher = deeper daylight penetration |
| Daylight depth rule | 2× window head height | Effective daylight depth |
| Side window limit | ~4.5m depth | Beyond this, supplementary needed |
| Rooflights | 3× more effective | Than same-area side windows |

## Orientation & Solar Gain

| Orientation | Solar Gain | Best For |
|-------------|-----------|----------|
| South | Maximum winter gain | Living rooms, solar collection |
| South-east | Morning + midday sun | Bedrooms, breakfast areas |
| East | Morning sun only | Bedrooms |
| West | Afternoon/evening sun | Caution: summer overheating |
| North | Minimal direct sun | Studios, offices (even light) |

## Natural Ventilation

| Parameter | Value | Notes |
|-----------|-------|-------|
| Openable window area | ≥ 1/20 of floor area | Minimum requirement |
| Cross-ventilation | Windows on 2 facades | Most effective method |
| Stack ventilation | High + low openings | Temperature-driven airflow |
| Single-sided limit | Room depth ≤ 2.5× ceiling height | ~6m for 2.5m ceiling |
| Min opening height | 1.75m above floor | For stack effect |
| Trickle vent size | 4000 mm² per room | Background ventilation |

## ModulCA Application
- 3×3m modules with 2.70m ceiling height allow daylight penetration across the full module depth (~5.4m rule)
- Corner modules can have windows on two facades for cross-ventilation
- L-shape and courtyard layouts maximize window-to-facade ratios
- Module flat roofs can accommodate skylights for interior modules
