# Furniture Clearances & Ergonomics

## Overview
Minimum space around furniture for comfortable use and circulation.

## Standards

| Element | Clearance | Notes |
|---------|-----------|-------|
| Bed side clearance | ≥ 600mm | One side min., 750mm both sides |
| Wardrobe front space | ≥ 800mm | Door swing + standing room |
| Kitchen work triangle | 3.6–6.6m total | Sink ↔ stove ↔ fridge |
| Kitchen counter depth | 600mm | Standard worktop |
| Kitchen aisle width | ≥ 900mm | ≥ 1200mm for two cooks |
| Dining chair space | ≥ 750mm | From table edge to wall |
| Toilet front clearance | ≥ 600mm | From bowl to wall/door |
| Shower min. size | 800 × 800mm | 900 × 900mm for comfort |
| Desk workspace depth | ≥ 700mm | With monitor at arm's length |
| Wheelchair turning | 1500mm ∅ | Accessible design requirement |

## ModulCA Application
With ~7m² usable space per 3×3m module, careful furniture placement following these clearances is essential. The module dimensions naturally accommodate standard bedroom furniture with proper clearances when using a single module, or generous living spaces when combining modules.
