# Corridors & Circulation

## Overview
Neufert standards for corridor widths and circulation spaces.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Minimum width | 0.90m | Absolute minimum |
| Recommended width | 1.10m | Comfortable passage |
| Accessible width | 1.20m | Wheelchair passage |
| Wheelchair turning | 1.50m × 1.50m | At direction changes |
| Max dead-end length | 7.50m | Fire regulation |

## ModulCA Application
In modular construction, corridors can be integrated into module layouts or created between module rows. The 3×3m module grid naturally creates circulation spaces when modules are arranged in L-shape, T-shape, or courtyard configurations.
