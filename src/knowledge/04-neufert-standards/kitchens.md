# Kitchen Design Standards

## Overview
Neufert standards for kitchen design, including the work triangle, counter dimensions, and ventilation.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Minimum area | 6m² (small) to 12m² (family) | Depends on household size |
| Work triangle (total) | ≤ 6.6m | Sink–stove–fridge |
| Work triangle (each leg) | 1.2–2.7m | Between appliances |
| Counter height | 85–90cm standard | 100–110cm bar height |
| Counter depth | 60cm | Standard worktop |
| Clearance between counters | min 1.20m | For two cooks: 1.50m |
| Ventilation extraction | min 60 l/s | During cooking |

## ModulCA Application
One 3×3m module (7m² usable) fits an L-shape kitchen comfortably. Two modules create a spacious kitchen-dining area with island. The work triangle is easily achieved within a single module, with sink and stove on the L-shape and fridge on the perpendicular wall.
