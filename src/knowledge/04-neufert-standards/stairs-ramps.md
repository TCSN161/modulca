# Stairs & Ramps

## Overview
Neufert standard dimensions for residential stairs and access ramps.

## Stair Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Rise (riser) | 170–190mm | Residential comfort range |
| Going (tread) | 260–290mm | Comfortable stride |
| Formula | 2×rise + going = 590–650mm | Blondel's formula |
| Min width | 0.90m (residential) | 1.20m for accessible |
| Headroom | min 2.10m | Clear above nosing line |
| Handrail height | 0.90–1.00m | Measured from nosing |
| Landing | Required every 18 steps max | Rest platform |

## ModulCA Application
In modular construction, stairs typically occupy one full 3×3m module for single-storey access. The module can include a straight flight, L-shape (quarter-turn), or U-shape (half-turn) stair depending on the layout configuration.
