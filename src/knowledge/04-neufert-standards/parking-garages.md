# Parking & Garage Standards

## Overview
Neufert dimensions for residential parking, garages, and driveways.

## Parking Space Dimensions

| Type | Width | Length | Notes |
|------|-------|--------|-------|
| Standard car | 2.50m | 5.00m | Open parking |
| Comfortable | 2.70m | 5.50m | Easy door opening |
| Accessible | 3.50m | 5.00m | Side transfer zone |
| Motorcycle | 1.00m | 2.50m | — |
| Bicycle | 0.60m | 2.00m | Per bike |

## Garage Dimensions

| Type | Width | Depth | Height | Notes |
|------|-------|-------|--------|-------|
| Single garage | 3.00m | 6.00m | 2.40m | Minimum internal |
| Single (comfortable) | 3.50m | 6.50m | 2.50m | Door opening space |
| Double garage | 6.00m | 6.00m | 2.40m | Two cars side by side |
| Garage door width | 2.50m (single) | — | 2.20m | Minimum clear opening |

## Driveway Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Min width | 2.75m | Single car |
| Comfortable width | 3.00m | Easy driving |
| Turning radius | 6.00m outer | For standard car |
| Gradient max | 15% (1:6.7) | Sloped drives |
| EV charging provision | 1 per dwelling | Future-ready |

## ModulCA Application
A single 3×3m module can serve as a carport or enclosed garage (9m² = 3.00 × 3.00m matches minimum single garage). Two modules side by side create a double garage. Pre-wire for EV charging (6mm² radial circuit) is standard in all ModulCA designs.
