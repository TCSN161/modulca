# Module Transport & Lifting

## Overview
Size limits, weight constraints, and crane requirements for moving modules to site.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Max module width (road) | 3.50m | Standard trailer, no escort |
| Max module height (road) | 4.20m | Including trailer deck |
| Max module length | 13.60m | Standard semi-trailer |
| Module weight (empty) | 2.5–4.0 t | 3×3m SIP/timber module |
| Crane capacity needed | ≥ 10 t | With rigging factor |
| Lifting points | 4 per module | ISO corner castings |
| Site access road width | ≥ 3.5m | For delivery truck |
| Assembly time per module | 1–2 hours | Crane + 4-person crew |
| Weather window | No rain / wind ≤ 35 km/h | For crane operations |

## ModulCA Application
Each 3×3m ModulCA module is designed to fit standard European road transport without special permits. The 4 ISO corner castings allow safe crane lifting. A typical single-family home (6-8 modules) can be assembled on-site in 1-2 days.
