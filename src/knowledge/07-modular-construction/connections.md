# Module-to-Module Connections

## Overview
How modules bolt together on site — structural, MEP, and weatherproofing joints.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Structural bolt grade | 8.8 or 10.9 | EN 14399 |
| Bolt spacing | 600mm c/c | Along shared wall |
| Gasket / seal | EPDM compressible strip | Weather + acoustic |
| Acoustic isolation | ≥ Rw 45 dB | Between dwelling units |
| Electrical jumper | Quick-connect plug | Pre-wired in factory |
| Plumbing jumper | Push-fit coupling | 15–22mm |
| Shared wall savings | €1,500 / wall | ModulCA discount |
| Alignment tolerance | ±5mm | On foundation grid |

## ModulCA Application
When two modules share a wall, the shared wall is removed to create open space. Module-to-module connections use high-strength bolts at 600mm centres, with EPDM gaskets for weather sealing and acoustic isolation. All MEP connections (electrical, plumbing) use quick-connect fittings pre-installed at the factory.
