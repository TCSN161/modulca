# Timber Frame Construction

## Overview
Specifications for CLT, glulam, and timber stud frame systems used in modular buildings.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| CLT panel thickness | 80–120mm | Walls; 120–200mm floors |
| Glulam beam class | GL24h / GL28h | EN 14080 certified |
| Timber stud spacing | 400–600mm c/c | Depends on load |
| Moisture content | ≤ 12% | Kiln-dried, EN 14298 |
| Fire treatment | Class B-s1, d0 | Euroclass for walls |
| Timber grade (structural) | C24 min. | EN 338 strength class |
| Design life | 50 years | EN 1990 Category 4 |
| Module transport max | 3.5 × 13.6m | Standard truck trailer |

## ModulCA Application
ModulCA modules use CLT or steel frame as primary structure. CLT offers superior thermal mass, carbon sequestration, and factory precision. Each 3×3m module is fabricated off-site with all structural elements pre-cut and assembled under controlled conditions.
