# Principles of Modular Construction

## Overview
Modular construction is a method where buildings are produced in discrete three-dimensional units (modules) in a factory, then transported and assembled on site. It fundamentally differs from traditional on-site construction.

## Core Principles

### 1. Standardized Grid
- ModulCA uses a **3.00 × 3.00m** module grid
- Each module = 9m² gross, ~7m² usable (after walls)
- Standard wall height: 2.70m clear interior
- Modules combine: line, L-shape, T-shape, U-shape, courtyard

### 2. Factory Production
| Advantage | Impact |
|-----------|--------|
| Weather-independent | No delays from rain/cold |
| Quality control | Consistent, inspected conditions |
| Waste reduction | < 5% material waste (vs. 15-20% traditional) |
| Speed | 60-70% faster than traditional |
| Safety | Controlled environment, fewer accidents |
| Precision | CNC cutting, ±1mm tolerance |

### 3. Design for Transport
- Max road width: 3.50m (no special permit)
- Max road height: 4.20m (including trailer)
- Max road length: 13.60m
- Module weight: 2.5–4.0 tonnes (3×3m)
- ISO corner castings for crane lifting

### 4. Design for Assembly
- All connections standardized (bolts, gaskets, jumpers)
- MEP quick-connects (electrical, plumbing)
- On-site assembly: 1-2 days for typical home
- Crane + 4-person crew

### 5. Design for Disassembly
- Bolted connections (not welded/glued)
- Screw pile foundations (removable)
- Modules can be relocated or reconfigured
- End-of-life: modules can be separated and recycled

## The 3×3m Module System

| Configuration | Usable Area | Typical Use |
|--------------|-------------|-------------|
| 1 module | 7m² | Bedroom, bathroom, kitchen, office |
| 2 modules (line) | 14m² | Living room, master bedroom |
| 3 modules (L-shape) | 21m² | Open-plan living/dining |
| 4 modules (square) | 28m² | Living/dining/kitchen open plan |
| 6 modules | 42m² | 1-bedroom apartment |
| 8 modules | 56m² | 2-bedroom house |
| 10-12 modules | 70-84m² | 3-bedroom family home |

## ModulCA Application
ModulCA embodies all five principles: standardized 3×3m grid, factory production with CLT/SIP, transport-optimized dimensions, quick-connect assembly, and reversible screw pile foundations.
