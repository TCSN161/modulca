# SIP Panels (Structural Insulated)

## Overview
Performance data for SIP wall and roof panels used in ModulCA modules.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Panel thickness | 120–200mm | Walls 120mm, roof 160mm |
| U-value (120mm) | 0.22 W/m²K | Meets nZEB in Romania |
| Core material | EPS / PUR | EN 14509 certified |
| Facing material | OSB-3 / cement board | 11–15mm each side |
| Compressive strength | ≥ 100 kPa | EN 826 |
| Fire rating | REI 60 | With gypsum board lining |
| Airtightness | ≤ 0.6 ACH @50Pa | When taped & sealed |
| Assembly speed | 1 module / day | 2-person crew |

## ModulCA Application
SIP panels are the standard wall system for ModulCA modules, providing excellent insulation in a thin profile. The factory-taped joints ensure consistent airtightness across all modules.
