# Foundation Systems

## Overview
Options for modular building foundations — screw piles, pads, and strip footings.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Screw pile diameter | 76–114mm | Shaft; helix 200–300mm |
| Screw pile depth | 1.5–3.0m | Below frost line |
| Bearing capacity | 25–50 kN/pile | Depends on soil |
| Frost depth (Romania) | 0.80–1.10m | Zone-dependent |
| Pad foundation size | 600 × 600mm | Min. for 3×3m module |
| Concrete class | C20/25 min. | EN 206 |
| Steel grade | S355 | Screw piles; EN 10025 |
| Installation time | 4–8 piles/hour | Mechanical driver |

## ModulCA Application
Screw piles are the preferred foundation for ModulCA modules — fast installation, minimal site disturbance, and fully reversible (can be unscrewed). Each 3×3m module typically requires 4 screw piles at the corners.
