# Prefab vs. Modular vs. Traditional

## Overview
Understanding the differences between construction methods helps clients make informed decisions.

## Method Comparison

| Aspect | Traditional (on-site) | Panelized (prefab) | Volumetric Modular |
|--------|----------------------|--------------------|--------------------|
| **Where built** | 100% on site | Panels in factory, assembled on site | Complete rooms in factory |
| **Factory completion** | 0% | 30-50% | 80-95% |
| **Site time** | 6-18 months | 3-9 months | 1-4 months |
| **Quality control** | Variable (weather, crew) | Moderate | High (factory-controlled) |
| **Waste** | 15-20% | 8-12% | 3-5% |
| **Cost** | €800-1,500/m² (RO) | €900-1,400/m² | €1,000-2,000/m² |
| **Design flexibility** | Maximum | High | Moderate (grid-based) |
| **Relocatable** | No | No | Yes (bolted connections) |
| **Weather dependency** | High | Moderate | Low |
| **Financing** | Staged payments | Staged | More upfront (factory) |

## Types of Prefab/Modular

| Type | Description | Example |
|------|-------------|---------|
| **Kit homes** | Pre-cut materials, assembled on site | Timber frame kits |
| **Panelized** | Wall/floor/roof panels, assembled on site | SIP panel homes |
| **Volumetric modular** | Complete 3D modules, stacked/connected | ModulCA system |
| **Hybrid** | Modular core + site-built additions | Core + extension |
| **Pod/unit** | Single function (bathroom pod, kitchen pod) | Bathroom pods for hotels |

## When Each Method is Best

| Choose This | When |
|-------------|------|
| **Traditional** | Complex bespoke design, unlimited budget, no time pressure |
| **Panelized** | Speed matters, but need more design flexibility than modular |
| **Volumetric modular** | Speed is critical, quality must be consistent, relocatability valued |
| **ModulCA** | All benefits of volumetric + 3×3m grid for flexible configurations |

## ModulCA Application
ModulCA uses **volumetric modular construction** on a standardized 3×3m grid. This provides the speed and quality advantages of full modular while offering significant design flexibility through module combination and configuration.
