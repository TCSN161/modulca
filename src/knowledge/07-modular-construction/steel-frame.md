# Light Steel Frame Construction

## Overview
Light Steel Frame (LSF) uses cold-formed steel C-sections and tracks as the structural frame for modular buildings. An alternative to timber frame with specific advantages.

## Specifications

| Parameter | Value | Notes |
|-----------|-------|-------|
| Steel grade | S350 GD+Z | EN 10346 (galvanized) |
| C-section depth | 89–150mm | Wall studs |
| Steel thickness | 1.0–2.0mm | Depending on load |
| Stud spacing | 400–600mm c/c | Standard modular |
| Galvanizing | Z275 (275 g/m²) | Corrosion protection |
| Design life | 60+ years | With proper galvanizing |
| Thermal bridge | Use thermal break strips | Between steel and cladding |
| Fire protection | 2×12.5mm Type F gypsum | Achieves REI 60 |

## Steel vs. Timber Frame

| Aspect | Light Steel Frame | Timber Frame |
|--------|-------------------|-------------|
| Weight | Lighter (20-30% less) | Moderate |
| Precision | Very high (CNC) | High (CNC) |
| Moisture resistance | Immune to rot | Requires treatment |
| Thermal bridging | Higher (needs breaks) | Lower (wood is insulating) |
| Carbon footprint | Higher embodied carbon | Carbon negative (CLT) |
| Fire behavior | Non-combustible frame | Combustible (needs protection) |
| Insect resistance | Immune | Needs treatment |
| Cost (Romania) | Similar | Similar |
| Recyclability | 95%+ recyclable | Reusable/recyclable |

## When to Choose Steel Frame
- High-humidity environments
- Termite/insect risk areas
- Long-span requirements
- Client preference for non-combustible structure
- Rapid factory production (no drying time)

## ModulCA Application
ModulCA offers both CLT and light steel frame options. Steel frame is recommended for modules exposed to high moisture (pool rooms, commercial kitchens) or where non-combustible structure is required by local fire codes.
