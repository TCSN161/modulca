# Landscaping & Gardens

## Overview
Neufert standards for outdoor spaces, gardens, and landscaping.

## Standards

| Element | Value | Notes |
|---------|-------|-------|
| Terrace/patio | min 12m² (3×4m) | South/south-west orientation preferred |
| Garden paths (single) | 0.60m | Walking path |
| Garden paths (double) | 1.20m | Two people side by side |
| Garden paths (wheelchair) | 1.50m | Accessible path |
| Planting distance (shrubs) | 0.50m from boundary | Property line setback |
| Planting distance (trees <3m) | 2.00m from boundary | Small trees |
| Planting distance (tall trees) | 4.00m from boundary | Large trees |
| Outdoor steps (rise) | 120–150mm | Lower than indoor |
| Outdoor steps (tread) | 350–400mm | Wider than indoor |
| Parking space | 2.50 × 5.00m | Per car |
| Accessible parking | 3.50m wide | Extra width for door opening |

## Romanian Regulations
- Min distance from side boundaries: 1.0m for structures
- Green space: typically 20–30% of lot required by Certificat de Urbanism

## ModulCA Application
Place modules to create courtyard/garden spaces. L-shape and U-shape layouts naturally frame outdoor living areas. South-facing terraces maximize solar gain for passive heating.
