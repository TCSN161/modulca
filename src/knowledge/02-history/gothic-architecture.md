# Gothic Architecture (1100 – 1500)

## Overview
Gothic architecture revolutionized building with the pointed arch, ribbed vault, and flying buttress — structural innovations that allowed walls to become thinner and windows larger. The result was architecture of unprecedented height and light, most famously expressed in the great cathedrals of Europe.

## Key Structural Innovations

| Innovation | Function | Impact |
|-----------|----------|--------|
| Pointed arch | Distributes load more efficiently than round arch | Allows varying span widths with consistent height |
| Ribbed vault | Structural ribs carry load, infill panels are thin | Lighter roof, easier construction |
| Flying buttress | External arch transfers lateral thrust | Walls can be thinner → more window area |
| Tracery | Decorative stone framework in windows | Enabled large stained glass windows |
| Pinnacle | Weighted top of buttress | Adds downward force to counteract thrust |

## Regional Variations

| Region | Character | Example |
|--------|-----------|---------|
| French Gothic | Height, structural logic | Chartres, Notre-Dame de Paris |
| English Gothic | Length, horizontal emphasis | Salisbury, Canterbury |
| German Gothic | Ornate, hall churches | Cologne Cathedral |
| Italian Gothic | Colored marble, less height | Florence Cathedral |
| Spanish Gothic | Width, Mudéjar influence | Seville, Burgos |
| Romanian/Moldavian Gothic | Painted exterior, fortress character | Curtea de Argeș, Moldavian monasteries |

## Gothic Principles Relevant Today

| Principle | Gothic Expression | Modern Application |
|-----------|------------------|-------------------|
| Structure as expression | Exposed ribs and buttresses | Exposed CLT structure, honest materials |
| Maximum daylight | Thin walls, large windows | Floor-to-ceiling glazing |
| Vertical aspiration | Soaring naves | Double-height living spaces |
| Craftsmanship | Stone carving, stained glass | Artisan timber detailing |
| Modular repetition | Bay system (repeated structural units) | 3×3m module grid |
