# Modernism (1920 – 1970)

## Overview
Modernism was the most transformative architectural movement of the 20th century. It rejected ornament and historical references in favor of function, technology, and social purpose. "Form follows function" became the creed. White walls, flat roofs, open plans, and industrial materials defined a new architectural language.

## Key Principles

| Principle | Description |
|-----------|-------------|
| Form follows function | Design driven by use, not decoration |
| Honest materials | Express structure and materials as they are |
| Open plan | Free-flowing interior spaces |
| Flat roof | Usable roof terrace, clean silhouette |
| Pilotis | Building raised on columns, freeing ground |
| Ribbon windows | Horizontal strip windows for even light |
| Industrial materials | Concrete, steel, glass |
| Mass production | Standardized components for affordable housing |

## Le Corbusier's Five Points (1926)

| Point | Description | ModulCA Parallel |
|-------|-------------|-----------------|
| Pilotis | Raise building on columns | Screw pile foundation |
| Free plan | Non-load-bearing interior walls | Open module interiors |
| Free facade | Facade independent of structure | Cladding options per module |
| Ribbon windows | Horizontal glazing bands | Full-width module windows |
| Roof garden | Flat roof as living space | Green roof / terrace modules |

| Architect | Country | Key Work | Contribution |

| Mies van der Rohe | Germany/USA | Barcelona Pavilion, Farnsworth House | "Less is more", universal space |

## Modernism and Modular Construction
Modernism and modular construction share DNA:

| Shared Value | Modernism | ModulCA |
|-------------|-----------|---------|
| Standardization | Standard components, mass production | Standard 3×3m module |
| Industrialization | Factory-made building parts | Factory-built complete modules |
| Social purpose | Housing for all, not just the rich | Affordable modular homes |
| Flat roof | Roof as usable surface | Green roof, PV, terrace |
| Honest materials | Exposed concrete, steel | Exposed CLT, timber |
| Minimal ornament | Clean lines, no decoration | Simple, functional design |

## ModulCA Connection
- **Le Corbusier's 5 Points**: ModulCA literally implements all five — pilotis (piles), free plan, free facade, ribbon windows, roof garden
- **The Modulor**: Le Corbusier's proportional system based on human body → ModulCA's 3m module width is close to his proportional grid
- **Bauhaus standardization**: Gropius dreamed of factory-built housing — ModulCA achieves it
- **Aalto's warmth**: Timber construction adds the humanism that brutalist concrete modernism lacked
