# Contemporary Architecture (2000 – Present)

## Overview
Contemporary architecture is characterized by pluralism — no single dominant style. Instead, multiple approaches coexist: sustainability-driven design, parametric computation, mass timber revolution, biophilic wellness, and social architecture. Technology enables forms previously impossible, while climate crisis drives a return to natural materials and energy-efficient design.

## Major Themes

| Theme | Description | Key Figures |
|-------|-------------|-------------|
| Sustainability | Net-zero, circular economy, bioclimatic | MVRDV, BIG, Snøhetta |
| Mass timber | CLT revolution, tall timber buildings | Shigeru Ban, Waugh Thistleton |
| Parametric | Algorithm-generated organic forms | Zaha Hadid Architects, Heatherwick |
| Biophilic | Nature-integrated, wellness-focused | WOHA, Kengo Kuma |
| Social | Affordable housing, community-focused | Alejandro Aravena (ELEMENTAL) |
| Adaptive reuse | Transforming existing buildings | Lacaton & Vassal (Pritzker 2021) |
| Modular/prefab | Factory-built, scalable | WikiHouse, ICON (3D printing) |
