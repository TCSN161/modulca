# Postmodernism & Deconstructivism (1970 – 2000)

## Overview
Postmodernism emerged as a reaction against Modernism's perceived coldness and uniformity. Where Modernism said "Less is more," Robert Venturi countered "Less is a bore." Postmodernism reintroduced color, ornament, historical references, and humor. Deconstructivism took rebellion further — fragmenting, tilting, and distorting architectural forms.

## Postmodernism (1970-1990)

| Feature | Description |
|---------|-------------|
| Historical references | Playful use of columns, pediments, arches |
| Color and ornament | Return of decoration after Modernist austerity |
| Context sensitivity | Buildings respond to their surroundings |
| Humor and irony | Architecture doesn't have to be serious |
| Mixed metaphors | Combining different styles and references |
| Populism | Architecture people enjoy, not just critics |

### Key Postmodernist Architects
| Architect | Key Work |
|-----------|----------|
| Robert Venturi | Vanna Venturi House, "Learning from Las Vegas" |
| Michael Graves | Portland Building, Disney buildings |
| Aldo Rossi | San Cataldo Cemetery, "Architecture of the City" |
| Charles Moore | Piazza d'Italia, Sea Ranch |
| James Stirling | Neue Staatsgalerie, Stuttgart |

## Deconstructivism (1988-2000s)

| Feature | Description |
|---------|-------------|
| Fragmentation | Buildings appear broken apart |
| Angular geometry | Sharp, non-orthogonal angles |
| Instability | Deliberate sense of unease |
| Material contrasts | Titanium, glass, raw concrete |
| Computational forms | Enabled by early CAD/CAM |

### Key Deconstructivist Architects
| Architect | Key Work |
|-----------|----------|
| Frank Gehry | Guggenheim Bilbao (1997) |
| Zaha Hadid | Vitra Fire Station, Heydar Aliyev Center |
| Daniel Libeskind | Jewish Museum Berlin |
| Rem Koolhaas | CCTV Beijing, Seattle Central Library |
| Coop Himmelb(l)au | UFA Cinema, Dresden |

## Lessons for Residential Design

| Lesson | Application |
|--------|-------------|
| Context matters | Design should respond to site, culture, climate |
| Users matter | Prioritize how people actually live |
| Rules can be broken thoughtfully | Module rotation, non-aligned layouts |
| Variety within system | Not every module needs to look the same |
| Fun is allowed | Architecture can bring joy |

## ModulCA Connection
- **Contextual design**: Module cladding and configuration can respond to local Romanian or Dutch building traditions
- **Variety in the grid**: Postmodernism teaches that a modular system can still be expressive — rotate a module, vary cladding, break the grid
- **User-centered**: Postmodernism's populism aligns with ModulCA's mission — homes designed for people, not theory
