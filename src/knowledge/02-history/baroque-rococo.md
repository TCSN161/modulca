# Baroque & Rococo (1600 – 1780)

## Overview
Baroque architecture amplified Renaissance classicism with drama, movement, and emotion. Walls curved, facades undulated, interiors overwhelmed with light and ornament. Rococo later refined this into lighter, more playful decoration. Both styles expressed power — of the Church, of absolute monarchs, of wealthy patrons.

## Baroque Characteristics

| Feature | Description | Example |
|---------|-------------|---------|
| Curved forms | Convex/concave facades, oval plans | Borromini's San Carlo (Rome) |
| Dramatic light | Concealed sources, chiaroscuro | Bernini's Ecstasy of St. Teresa |
| Grand scale | Monumental axes, vast interiors | Palace of Versailles |
| Rich materials | Marble, gold, fresco | St. Peter's Basilica interior |
| Integration of arts | Architecture + sculpture + painting | Total work of art (Gesamtkunstwerk) |
| Axial planning | Long vistas, symmetrical gardens | Versailles gardens |

## Rococo Characteristics

| Feature | Description | Example |
|---------|-------------|---------|
| Lighter palette | Pastels, white, gold | Amalienburg, Munich |
| Asymmetric ornament | C-scrolls, shells, flowers | Interior decoration |
| Smaller scale | Salons, boudoirs, pavilions | Intimate spaces |
| Nature motifs | Flowers, vines, birds | Decorative plasterwork |
| Playfulness | Wit, charm over grandeur | Garden follies |
