# Neoclassical Architecture (1750 – 1850)

## Overview
Neoclassicism was a deliberate return to "pure" Greek and Roman forms, reacting against Baroque excess. Driven by Enlightenment rationalism and archaeological discoveries (Pompeii, Paestum), it became the architecture of democracy, institutions, and emerging nation-states.

## Key Characteristics

| Feature | Description |
|---------|-------------|
| Geometric purity | Simple volumes — cubes, cylinders, spheres |
| Greek temple front | Pediment + columns = authority |
| Symmetry | Strict bilateral symmetry |
| Minimal ornament | Classical orders used sparingly |
| White/pale stone | Clean, rational aesthetic |
| Monumental stairs | Processional entrance |

## Neoclassical in Romania
Romania's institutional architecture from the 19th century is largely Neoclassical:

| Building | Architect | Year |
|----------|-----------|------|
| Romanian Athenaeum | Albert Galleron | 1888 |
| National Bank of Romania | Cassien Bernard | 1885 |
| Justice Palace, Bucharest | Albert Ballu | 1895 |
| University of Bucharest | Alexandru Orăscu | 1857 |
