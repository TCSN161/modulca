# Art Nouveau (1890 – 1910)

## Overview
Art Nouveau ("New Art") broke from historical styles entirely, drawing inspiration from natural forms — plants, flowers, flowing water, organic curves. It was the first truly modern movement, treating the building as a total work of art where architecture, furniture, decoration, and craftsmanship formed a unified whole.

## Key Characteristics

| Feature | Description |
|---------|-------------|
| Organic curves | Whiplash lines, plant-like forms |
| Nature motifs | Flowers, vines, insects, birds |
| Total design | Architecture + interiors + furniture as one |
| New materials | Iron, glass used expressively |
| Craftsmanship | Handmade details, artisan quality |
| Asymmetry | Natural, flowing compositions |
