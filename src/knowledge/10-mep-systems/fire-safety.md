# Fire Safety (General)

## Overview
Fire ratings, detection, and escape requirements for modular dwellings.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Wall fire rating | REI 60 | With gypsum board lining |
| Floor fire rating | REI 60 | Between storeys |
| Smoke detection | Optical + heat | Each room + hallway |
| Escape distance | ≤ 9m | To exit door (single direction) |
| Exit door width | ≥ 800mm | Clear opening |
| Emergency lighting | 1 lux on escape route | 3-hour battery backup |
| Fire extinguisher | 1 per dwelling | ABC powder, 6 kg |
| Cladding reaction | Class B-s2, d0 | Euroclass minimum |

## ModulCA Application
Each ModulCA module achieves REI 60 fire rating through factory-installed gypsum board lining. Smoke detectors and emergency lighting are pre-wired during factory assembly. Module-to-module connections include fire stops at all penetrations.
