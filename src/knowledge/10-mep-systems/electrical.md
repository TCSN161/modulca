# Electrical Installation

## Overview
Wiring standards, outlet counts, and circuit sizing for modular residential units.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Supply voltage | 230V / 50Hz | Single phase (Romania) |
| Main breaker | 32A per module | MCB in consumer unit |
| Ring circuit | 2.5mm² T&E | Socket outlets |
| Lighting circuit | 1.5mm² T&E | Max 10 points / circuit |
| Socket outlets (bedroom) | 4–6 | Double sockets count as 2 |
| Socket outlets (kitchen) | 6–8 | Above counter + appliance |
| RCD protection | 30mA | All circuits (NP I7) |
| Cable route | In stud / behind lining | Concealed wiring |
| EV charger prep | 6mm² radial | Future-proofing |

## ModulCA Application
All electrical wiring is pre-installed in the factory, with quick-connect jumpers at module boundaries. Each module has its own sub-circuit connecting to the main consumer unit. EV charger preparation is included as standard future-proofing.
