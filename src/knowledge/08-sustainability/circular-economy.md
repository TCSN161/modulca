# Circular Economy in Construction

## Overview
The construction industry produces 35% of EU waste. The circular economy model aims to eliminate waste by designing buildings that can be maintained, adapted, disassembled, and reused. Modular construction is inherently aligned with circular principles — especially when using bolted (reversible) connections.

## Circular Design Strategies

| Strategy | Description | Impact |
|----------|-------------|--------|
| Design for Disassembly (DfD) | Reversible connections, avoid adhesives | Enables component reuse |
| Design for Adaptability | Flexible layouts, demountable partitions | Extends building life |
| Material passports | Document what's in each component | Enables future material recovery |
| Modular construction | Standardized, factory-built units | Reusable, relocatable |
| Lean manufacturing | Minimize factory waste | Less material sent to landfill |
| Bio-based materials | Timber, hemp, straw | Renewable, biodegradable |
| Recycled content | Recycled steel, recycled insulation | Reduces virgin material demand |

## Circularity Levels

| Level | Description | Example |
|-------|-------------|---------|
| 1. Building reuse | Entire building repurposed | Office → housing conversion |
| 2. Building relocation | Entire building moved | ModulCA modules to new site |
| 3. Component reuse | Elements reused as-is | Windows, doors, modules |
| 4. Material recycling | Processed into new products | Steel recycled, concrete crushed |
| 5. Energy recovery | Incineration with energy capture | Timber waste → biomass energy |
| 6. Disposal | Landfill (worst case) | Contaminated materials |

## Connection Types and Reversibility

| Connection | Reversible? | Reuse Potential | Examples |
|------------|------------|-----------------|---------|
| Bolts/screws | Yes | High — component reuse | Steel connections, module-to-module |
| Mechanical clips | Yes | High | Raised floor systems, facade panels |
| Mortar | No | Low — material recycling only | Brick walls |
| Welding | No | Low — requires cutting | Steel frames |
| Adhesive | No | Very low | Glued finishes |
| Concrete cast | No | Low — crushing only | Foundations, slabs |

## Dutch Circular Construction (Circulair Bouwen)

The Netherlands is a European leader in circular construction:

| Initiative | Description |
|-----------|-------------|
| Platform CB'23 | Industry agreement on circular construction definitions |
| MilieuPrestatie Gebouwen (MPG) | Environmental performance requirement for new buildings |
| MPG limit (2025) | ≤ 0.8 per m² floor area per year | Required for building permit |
| Madaster | Material passport platform — register building materials |
| Government target | 50% circular construction by 2030 |
