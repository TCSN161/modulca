# Residential Water Management

## Overview
Sustainable water management at the residential scale includes rainwater harvesting, greywater recycling, efficient fixtures, and on-site drainage. In the Netherlands this is critical (waterschap requirements). In Romania it's increasingly important for nZEB compliance and rural self-sufficiency.

## Household Water Consumption

| Use | Liters/person/day | % of Total | Potable Needed? |
|-----|-------------------|-----------|----------------|
| Toilet flushing | 30-40 | 27% | No |
| Shower/bath | 35-50 | 33% | Yes |
| Washing machine | 15-20 | 13% | No |
| Kitchen/drinking | 10-15 | 10% | Yes |
| Garden irrigation | 10-30 | 10% | No |
| Cleaning | 5-10 | 7% | No |
| **Total** | **105-165** | **100%** | **~43% must be potable** |

## Rainwater Harvesting

| Parameter | Value | Notes |
|-----------|-------|-------|
| Roof collection rate | ~0.8 l/m²/mm rainfall | Flat roof, 80% runoff coefficient |
| Annual rainfall (Romania) | 500-700mm | Continental climate |
| Annual rainfall (Netherlands) | 800-900mm | Maritime climate |
| Yield (100m² roof, RO) | 40,000-56,000 l/year | Enough for toilet + garden |
| Yield (100m² roof, NL) | 64,000-72,000 l/year | Ample for non-potable uses |
| Tank size (household) | 2,000-5,000 liters | Underground or utility room |
| Filter requirement | First-flush diverter + mesh | Pre-tank filtration |
| Uses | Toilet, washing machine, garden | Non-potable |

## On-Site Drainage (SuDS)

| System | Function | Size Guide |
|--------|----------|------------|
| Rain garden | Infiltration + biodiversity | 2-5% of roof area |
| Infiltration crate | Below-ground storage + infiltration | 20-60 l/m² impervious area |
| Permeable paving | Surface infiltration | Driveway, paths |
| Green roof | Retention + evapotranspiration | 25-40 l/m² capacity |
| Swale | Conveyance + infiltration | Along property boundary |

## ModulCA Application
- **Flat roof collection**: Each module collects rainwater → central tank
- **Green roof modules**: 25-40 l/m² retention — meets Dutch waterschap requirements
- **Pre-plumbed rainwater**: Factory installs dual plumbing (potable + rainwater) in bathroom and utility modules
- **Infiltration crates**: Below module-free areas of plot — absorb overflow
- **Permeable surroundings**: Modules on screw piles leave maximum ground area permeable
- **Efficient fixtures**: Factory-fitted water-efficient taps, dual-flush toilets, low-flow showerheads
