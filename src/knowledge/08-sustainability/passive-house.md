# Passive House & Sustainability Principles

## Overview
Key principles for achieving passive house performance and sustainability in modular construction.

## Standards

| Parameter | Value | Notes |
|-----------|-------|-------|
| Heating demand | ≤ 15 kWh/m²·year | Passive House standard |
| Orientation | Living areas south/SE | Maximize solar gain |
| Wall insulation (U-value) | ≤ 0.20 W/m²K | Walls |
| Roof insulation (U-value) | ≤ 0.15 W/m²K | Roof |
| Windows | Triple glazing, U ≤ 1.0 | High-performance glazing |
| Air tightness | n50 ≤ 0.6/h | Blower door tested |
| Natural ventilation | Cross-ventilation | Windows on opposite walls |
| Solar panels | min 3 kWp recommended | For a modular home |

## CLT Advantage
Cross-Laminated Timber (CLT) is a carbon-negative material with excellent thermal mass. Factory precision ensures consistent air tightness, which is critical for passive house performance.

## ModulCA Application
ModulCA modules are designed with passive house principles in mind:
- Factory-controlled airtightness (tested before delivery)
- Pre-installed insulation exceeding nZEB requirements
- Flat roof accommodates 8m² PV per module
- CLT structure provides thermal mass and carbon sequestration
