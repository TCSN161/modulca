# U-Value Requirements

## Overview
Maximum U-values for building elements per Romanian C107 and nZEB standards.

## Standards

| Element | C107-2005 | nZEB Target | ModulCA Actual |
|---------|-----------|-------------|----------------|
| External wall | ≤ 0.28 W/m²K | ≤ 0.22 W/m²K | 0.18 W/m²K |
| Roof / ceiling | ≤ 0.20 W/m²K | ≤ 0.15 W/m²K | — |
| Ground floor | ≤ 0.35 W/m²K | ≤ 0.25 W/m²K | — |
| Windows | ≤ 1.30 W/m²K | Triple glazing recommended | — |
| External doors | ≤ 1.80 W/m²K | Insulated core | — |
| Thermal bridges | Ψ ≤ 0.05 W/mK | Junction details | — |

## ModulCA Application
ModulCA wall assemblies achieve 0.18 W/m²K, exceeding both the C107-2005 requirement and the nZEB target. Factory-controlled construction ensures consistent insulation performance with minimal thermal bridging at junctions.
