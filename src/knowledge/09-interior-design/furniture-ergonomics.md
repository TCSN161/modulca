# Furniture & Ergonomics

## Overview
Ergonomics — the study of how humans interact with their environment — determines furniture dimensions and placement. In modular homes where space is optimized, correct furniture sizing and clearances are essential to make every module comfortable and functional.

## Compact Living Strategies

| Strategy | Description | Space Saved |
|----------|-------------|-------------|
| Wall bed (Murphy bed) | Folds up into wall | 3-4m² of floor space |
| Sofa bed | Living by day, guest bed by night | Eliminates guest bedroom |
| Extendable dining table | 4-seat becomes 6-8 | Daily: compact, guests: expanded |
| Built-in bench seating | Storage inside bench | Saves chairs + storage furniture |
| Stacking chairs | Store vertically when not needed | Flexible dining/workspace |
| Over-door hooks | Use vertical dead space | No floor space used |
| Under-bed storage | Drawers or lift-up base | Full use of bed footprint |

## ModulCA Application
- **Built-in furniture**: Factory-install wardrobes, kitchen, bathroom vanity — no furniture delivery needed
- **Multi-function modules**: Murphy bed in office module = guest room when needed
- **Module-sized furniture**: Design furniture to match 3m module width — custom fit
- **Bench seating**: Built-in window bench across module window wall — seating + storage
- **Modular shelving**: Wall-mounted systems that attach to CLT walls (no drilling into drywall)
- **Clearance guarantee**: Factory layout ensures all furniture clearances are met before delivery
