# Storage Solutions for Compact Homes

## Overview
Storage is the most common complaint in compact homes. The solution isn't more space — it's smarter storage design. Built-in storage, multi-function furniture, and vertical thinking can provide ample storage in a modular home without wasting living space.

## Storage Requirements per Room

| Room | Min Storage | Recommended | Type |
|------|-----------|-------------|------|
| Master bedroom | 1.2m wardrobe | 2.4m+ wardrobe + drawers | Built-in sliding door |
| Children's bedroom | 1.0m wardrobe | 1.8m wardrobe + shelving | Adjustable height |
| Entrance | 0.5m coat area | 1.2m coat + shoe + shelf | Closed cabinet |
| Kitchen | 3.0 linear m cabinets | 5.0+ linear m | Upper + lower + pantry |
| Bathroom | Vanity + 1 shelf | Vanity + mirror cabinet + linen closet | Moisture-resistant |
| Living room | Bookshelf or media unit | Wall-to-wall shelving system | Open or closed |
| Home office | Desk storage + bookshelf | Full wall shelving + filing | Functional layout |
| Utility/laundry | Appliance space + shelving | Closed cabinets + hanging | Concealed if possible |

## Built-In Storage Strategies

| Strategy | Description | Where |
|----------|-------------|-------|
| Floor-to-ceiling wardrobes | Use full wall height (2.70m in ModulCA) | Bedroom, entrance |
| Under-stair storage | Drawers or closet under stairs | If 2-story module |
| Window seat with storage | Bench with lift-up or drawer base | Living room, bedroom |
| Raised platform bed | Full storage underneath (300mm+ height) | Bedroom |
| Kitchen ceiling cabinets | Cabinets to ceiling height | Kitchen |
| Recessed niches | In-wall shelving (non-structural walls) | Bathroom, bedroom |
| Over-door shelving | Shelf above doorframe | Any room |
| Module perimeter storage | 300mm-deep cabinets along one wall | Any room |

## Storage per Household Member

| Category | Space Needed | Notes |
|----------|-------------|-------|
| Clothing (adult) | 1.0-1.5m hanging + 0.5m³ folded | Per person |
| Shoes (adult) | 10-20 pairs = 0.3m³ | Entrance or bedroom |
| Books | 0.5-2.0 linear m shelving | Per reader |
| Kitchen items (couple) | 3-4 linear m cabinet | Utensils, food, cookware |
| Cleaning supplies | 0.5m³ dedicated closet | Utility or kitchen |
| Seasonal items | 1-2m³ per household | Attic, external storage |
| Hobby/sports | Variable | Dedicated storage zone |

## ModulCA Application
- **2.70m height advantage**: ModulCA's ceiling height allows tall built-in storage — 30% more than standard 2.40m rooms
- **Factory-built wardrobes**: Sliding-door wardrobes installed in factory — arrive ready
- **Module perimeter storage**: 300mm-deep cabinets along bedroom module walls — hidden yet accessible
- **External storage module**: One 3×3m module as dedicated storage/utility — garden tools, bikes, seasonal items
- **CLT wall advantage**: Screw directly into CLT for shelving — no hollow-wall anchors needed
- **Modular closet systems**: Standardized insert systems that fit any module — relocate as needs change
