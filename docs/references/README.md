# ModulCA Reference Library

Drop images, links, notes and examples in each folder to guide development.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `step-01-land/` | Map UI, address search, ANCPI integration, terrain selection |
| `step-02-design/` | Module placement, grid, drag & drop, project layers |
| `step-03-preview/` | 3D preview, terrain, camera controls |
| `step-04-style/` | Style direction, moodboard, Pinterest reference |
| `step-05-configure/` | Wall types, floor plans, module customization |
| `step-06-visualize/` | 3D interior, furniture placement, rotation, layouts |
| `step-07-render/` | AI rendering, camera angles, atmosphere, materials |
| `step-08-technical/` | Drawings, sections, elevations, details, PDF export |
| `step-09-walkthrough/` | First-person navigation, VR-like experience |
| `step-10-products/` | Product catalog, prices, retailers, categories |
| `step-11-finalize/` | Summary, export, project completion |
| `ui-ux/` | General UI/UX patterns, competitor screenshots, inspiration |
| `3d-models/` | 3D furniture models (glTF/GLB), sources, licenses |
| `branding/` | Logo, colors, typography, brand guidelines |

## How to Use

1. Drop screenshots, photos, or PDFs in the relevant folder
2. Add a `notes.md` in any folder with written descriptions
3. Include URLs to reference websites
4. Claude will read these when working on that step

## 3D Model Sources (Free)

- **Sketchfab** (sketchfab.com) — Best API, glTF native, 500K+ free models
- **Poly Pizza** (poly.pizza) — Free REST API, all glTF, low-poly style
- **Objaverse** (objaverse.allenai.org) — 800K+ models, CC licensed
