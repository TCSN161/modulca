## Technical Requirements
- Format: glTF 2.0 (.glb preferred for single file)
- Max file size: 5MB per model (for web performance)
- License: CC0 or CC-BY (commercial use allowed)
- Scale: Real-world dimensions in meters
- Origin: Center bottom of object
