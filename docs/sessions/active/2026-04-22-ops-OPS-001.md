   → **save this as `TELEGRAM_BOT_TOKEN`**

   → **save that number as `TELEGRAM_CHAT_ID`**

   - Name: `TELEGRAM_BOT_TOKEN`, Value: (the token from step 1)
   - Name: `TELEGRAM_CHAT_ID`, Value: (the chat_id from step 2)
